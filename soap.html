Запуск тестов и отчёты

Через UI: правой кнопкой на Test Suit`e > Launch Test Runner. Далее идем в Reports и активируем нужные чекбоксы:
- экспортировать результат тестов в Junit-style в xls формате, чтобы сгенерировать html нужны танцы с бубном (или есть удобное решение о котором мне неизвестно)
- выводить результаты всех шагов во всех кейсах
- напечатать краткий отчёт в консоль SoapUI
Плюсы запуска через UI - нет порога входа, чекбокс краткого отчёта легко показывает соотношение pass/fail кейсов.
Минусы - невозможно собирать статистику по прогонам тестов, сложно анализировать завалившиеся кейсы.
