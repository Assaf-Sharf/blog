# 1.CSS이란

CSS는 HTML이나 XML과 같은 구조화 된 문서를 화면, 종이 등에 어떻게 렌더링할 것인지를 정의하기 위한 언어이다.

HTML의 tag을 쓰지 않고 굳이 CSS의 언어를 써야하는 이유는, HTML의 tag는 변경해야될 때마다 모든 HTML을 변경해야되지만 CSS의 언어를 사용함으로써 중복코드를 없애고 많은 태그들을 한번에 변경가능해서 더욱 효율적으로 표현이 가능하기 때문이다.

------

# 2. CSS 적용 방법

### 1. 내부 스타일시트_ Internal

html 파일에 스타일을 기술하는 방법이다.

```
<head>
<style>
a{
  color:red;
}
</style>
</head>
```

: 스타일 태그를 사용함으로써 모든 a tag는 빨간색으로 변경된다고 CSS언어로 해석된다. (세미콜론은 효과를 끝나는 지점에 항상 붙여주어야 함!!!!)

**• CSS 구문은 선택자(selector)와 선언부(declaration)로 구성된다.**

> a =>선택자(selector) : 무엇을 꾸밀지 정한다.
> color =>속성(property) : 어떤 모양을 꾸밀지 정한다.
> red =>값(vlue) : 어떻게 꾸밀지 정한다.
> color:red; =>선언(declaration)

### 2.외부 스타일시트_External

별도의 파일에 CSS문서를 작성하고 해당 CSS를 필요로 하는 html 문서에서 불러와 사용하는 형식이다.(외부파일 확장자는 .CSS)

```
<link rel="stylesheet"  href="mystyle.css">
```

### 3.인라인 스타일_lnline

html 태그에 필요한 디자인 속성을 직접 작성하는 형식이다.

```
<a href=“1.html" style="color:red">CSS</a>
```

: 해당 a tag는 글씨가 빨강색으로 변경된다.

style속성은 CSS효과를 나타내는 것으로, 속성의 값을 CSS문법에 따라 해석하여 태그에 적용하는 것이다.
이때, “color:red”는 HTML의 언어이다.

• 스타일 순위
브라우저 디자인 정의 -> 외부 스타일시트 -> 내부 스타일시트-> 인라인 스타일시트

# 3.CSS 선택자(selector)

### 1.class 선택자

```
.saw{
  color:gray;
}
.active{
  color:red;
}

<a href="HTML.html" class="saw active">HTML</a>
<a href="CSS.html" class="saw">CSS</a>
```

:HTML은 빨강색으로 표시되고, CSS는 회색으로 표시된다.

> .saw => class가 saw인 tag을 선택.
> .active => class가 active인 tag을 선택.

• class는 tag들을 하나로 묶을 수 있으며, 여러개의 값을 선택자로 사용하고 선택자를 통해 기능을 추가할 수 있다.
• class의 단점은 class의 속성에 여러개의 선택자를 넣을 수 있는데 CSS의 선택자의 속성이 중복이 되면, 최근에 작성된 선택자의 값만 설정된다!!!

### 2.id 선택자

```
#active{
  color:black;
}
.saw{
   color:red;
}

<a href="CSS.html" class="saw" id=“active”>CSS</a>
```

: CSS는 빨강색으로 표시된다. (최근에 작성된 saw가 설정되지 않고 active의 속성이 설정된다!!)

<CSS : 선택자의 순위>
-선택자 : id>class>tag
-id의 속성의 값을 중복되어서는 안됨.
-선택자순위가 있는 이유는 전체적인 tag selector을 설정하고 id로 세부적인 디자인을 하기 위해서 순위를 생성함.

# 4.CSS box model

•block level element = 화면 양옆 전체를 자신의 영역으로 사용하는 태그
•inline element = 자기 자신의 크기만큼의 영역을 차지하는 태그

-아무런 의미 없지만 디자인을 위한 태그
•div : 블록레벨
•span : 인라인

### 1.box model 테두리설정

```
<style>
    h1{
      border:5px solid red;
      padding:20px;
      margin:0;
      dispaly:block;
      width:100px;
    }
</style>
```

•padding = 글씨와 테두리에 여백을 조절할 때
•margin = 테두리와 테두리의 간격을 조절할 때
•border = 내용에 padding을 더한 영역의 경계를 나타내며 테두리 역할을 한다.
•width = 내용의 가로길이를 표시
•border-width:5px; = 테두리 두께 5px
•border-color:red; = 테두리 빨강색
•border-style:solid; = 테두리 실선
= border: 5px red solid;

box-sizing 속성을 통해 쉽게 비율조정가능

### 2. grid 문법

grid선택자두개의 단락을 나눌 때, 문단1, 문단2는 각각의 태그로 묶어져 있어야 함(div tag 자주쓰임)
그 후에 문단1, 문단2는 하나의 부모태그로 묶여야 되는데 이때 선택자 grid를 사용함

```
<style>
    #grid{
      display:grid;
      grid-template-columns: 150px 1fr;
    }
    div{
      border: 5px solid gray;
    }
    </style>

<div id="grid">
<div>NAVIGATION</div>
<div>ARTICLE</div>
</div>
```

:왼쪽으로 NAVIGATION이 150px 차지하고, 오른쪽으로 ARTICLE이 1fr크기 차지한다.

• grid-template-columns 속성으로 열의 개수와 폭을 정합니다.
ex) grid-template-columns: 100px 50% auto; = 3열이 만들어지고, 1열의 너비는 100px, 2열의 너비는 50%, 3열의 너비는 남은 공간 전부가 됩니다.

•grid-template-rows 속성으로 행의 높이를 정합니다.
ex) grid-template-rows: 80px 120px; = 첫번째 행이 80px, 두번째행이 120px로 만들어집니다.

# 5.CSS 반응형 디자인_mediaqurey

화면의 크기에 따라서 웹페이지의 요소들이 변경됨

```
@media(min-width:800px){
  div{
    display:none;
  }
  }

<body>
  <div>
    responsive
    </div>
</body>
```

: 화면의 크기가 800px보다 크다면 responsive글씨를 없앤다.

@media(min-width:800px)=>스크린의 크기가 800px보다 크다면 div태그(responsive)를 없앤다
@media(max-width:800px)=>스크린의 크기가 800px보다 작다면 div태그(responsive)를 없앤다
