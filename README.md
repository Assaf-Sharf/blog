# Cisco Metapod Documentation

Metapod is a production-ready, OpenStack-based, on-premises solution that Cisco engineers deploy and remotely operate 24x7x365 on your behalf.

## [Docs Installation and Setup](https://mmistakes.github.io/minimal-mistakes/docs/installation/)

### How to Write and Build

- Write with Markdown in the ``_pages`` directory
- Site theme is compatible with Jekyll 3.x
- Workflow is to git clone, create and work in a branch, push up a Pull Request (PR), merge to master, then publish by pushing master to ``gh-pages`` branch.
- Build locally with ``bundle install``
- Docs pages use the single layout
- Sidebar configuration is found in the ``_config.yml`` file

#### Guides

- User Guide - written for end-users consuming cloud resources from Metapod such as images, volumes, Orchestration stacks, and networks
- Administrator Guide - written for cloud administrators who manage users, roles, projects, capacity, quotas, networks, and support end-users of Metapod
