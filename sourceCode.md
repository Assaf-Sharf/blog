The BIG IoT API and Marketplace are currently being built as part of the [BIG IoT EU project](http://big-iot.eu).
We are in the process of making the source code available to the public at the Eclipse Foundation. You can find our Eclipse project proposal [here](https://projects.eclipse.org/proposals/big-iot).
