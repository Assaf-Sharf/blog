# 개행문자(Newline) 설정

- 줄바꿈 처리해주는 문자: 운영체제마다 조금씩 다를 수 있음!
- 우리 운영체제가 아닌 다른 운영체제에서도 잘 돌아가게끔 하기 위해서!

```
git config --global core.autocrlf input
```
