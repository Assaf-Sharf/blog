# 함수 복습

- 기명 함수 (함수 선언)

```js
function sum(x, y) {
  return x + y;
}
```

- 익명 함수 (함수 표현)

```js
const sum = function (x, y) {
  return x + y;
};
```

- return === 함수 종료
- 그냥 return이면 **undefined**

```js
function sum(x, y) {
  if (x < 2) return;

  return x + y;
}

console.log(sum(1, 3)); // undefined
```

# 화살표함수

## ( ) => { } vs function ( ) { }

- 매개변수 2개 이상 : 소괄호 생략 불가능

```js
const double = function (x, y) {
  // 매개변수 2개 이상일땐 소괄호 생략 불가능
  return x * 2;
};
console.log("double: ", double(7));
```
