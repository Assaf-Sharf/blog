# 정규표현식(RegExp)

정규식, Regular Expression

# 생성자 방식

```js
const regexp = new RegExp("the", "gi");
```

# 리터럴 방식

```js
const regexp = /the/gi;
```

## 정규식.test(문자열) : 문자 검색

- 일치 여부(Boolean) 반환

```js
const regexp = /fox/gi;
console.log(regexp.test(str));
// true
```

<br>

## 문자열.match(정규식) : 문자 추출

- 일치하는 문자들을 담은 배열(Array) 반환

```js
const regexp = /the/gi; // global, ignoreCase
console.log(str.match(regexp));
// Array(3)
// 0: 'the'
// 1: 'The'
// 2: 'the'
```

<br>

## 문자열.replace(정규식) : 문자 대체

- 일치하는 문자를 대체한 결과 반환
- 원본문자열을 수정 x

```js
const regexp = /fox/gi;
```
