# <span class="half_HL">✔️ 형광펜 밑줄 효과</span>
- ```linear-gradient``` 는 직선으로 진행하는 색상 무늬를 생성합니다.
- ```to top``` : 방향을 to로 지정할 수 있는데, top으로 설정함으로써 위에서 아래로 진행될 수 있게 설정했습니다.
- ```색상 40%``` : 퍼센트를 조절하여 밑줄 효과의 두께를 조절할 수 있습니다.
- ```transparent 0%``` : 번지는 현상을 없애기 위해 0%로 설정했습니다. 

## 1. 바로 적용하기

오늘의 명언 부분에 형광펜 밑줄 효과를 추가해보겠습니다.

```markdown
<span style=" background: linear-gradient(to top, #F9E79F 40%, transparent 0%)" >오늘의 명언</span> 
: 책 한 권 읽기를 간절히 바라는 사람과 읽을 만한 책을 기다리다 지친 사람 사이에는 매우 큰 차이가 있다.
```

<span style=" background: linear-gradient(to top, #F9E79F 40%, transparent 0%)" >오늘의 명언</span> : 책 한 권 읽기를 간절히 바라는 사람과 읽을 만한 책을 기다리다 지친 사람 사이에는 매우 큰 차이가 있다.

## 2. CSS 파일에 저장하여 사용하기

CSS로 만들어놓고 class를 이용해 편리하게 형광펜 효과를 적용시킬 수 있습니다.

### (1) _sass > minimal-mistakes > base.scss

```css
/* 내가 추가한 것 : 형광펜 밑줄 기능 */
.half_HL {background: linear-gradient(to top, #EF9DA1 40%, transparent 0%);}
```

위 코드를 복사하여 base.scss 파일에 붙여넣으면 됩니다.
제가 형광펜 밑줄 기능으로 설정해둔 색상은 ```#EF9DA1```로 진한 핑크색입니다. 원하는 색상으로 변경하여 저장하면 됩니다.

위치는 어디든 상관없으나, 저는 98번 줄에 넣었습니다. [👉 Github 에서 변경사항 확인하기](https://github.com/j-jae0/j-jae0.github.io/commit/915e444dafa6729bee6307c90eb13fe75223197e)

만약, 형광펜 밑줄 기능을 색상별로 저장하고 싶다면 다음과 같이 여러 개의 클래스를 만들면 됩니다.

```css
/* 형광펜 : 노란색 */
.y_half_HL {background: linear-gradient(to top, #F7DC6F 40%, transparent 0%);}
/* 형광펜 : 빨간색 */
.r_half_HL {background: linear-gradient(to top, #CD6155 40%, transparent 0%);}
/* 형광펜 : 파란색 */
.b_half_HL {background: linear-gradient(to top, #7FB3D5 40%, transparent 0%);}
/* 형광펜 : 초록색 */
.g_half_HL {background: linear-gradient(to top, #58D68D 40%, transparent 0%);}
/* 형광펜 : 보라색 */
.p_half_HL {background: linear-gradient(to top, #AF7AC5 40%, transparent 0%);}
```

### (2) markdown 에 적용하기

css 파일에 저장해 둔 형광펜 밑줄 기능은 아래와 같이 사용하면 됩니다.

```markdown
<span class="y_half_HL">노란색 형광펜 밑줄</span>
<span class="r_half_HL">빨간색 형광펜 밑줄</span>
<span class="b_half_HL">파란색 형광펜 밑줄</span>
```
