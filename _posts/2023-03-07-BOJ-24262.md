## 문제  
24262

## solution  
```
MenOfPassion(A[], n) {
    i = ⌊n / 2⌋;
    return A[i]; # 코드1
}
```
입력의 크기 n이 주어지면, 알고리즘 수행 시간을 예제 출력과 같은 방식으로 출력해보자.

## CODE  

```python
print(1,0,sep='\n')
```
첫째 줄에 코드1의 수행 횟수를 출력한다. 둘째 줄에 코드 1의 수행 횟수를 다항식으로 나타내었을때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.  
문제의 코드는 단순히 인자로 배열 A와 n을 입력받고 특정 원소를 반환하는 함수다. index에 접근하는것은 O(1) 이므로 상수 시간이 소요된다. 어떤 n이 들어와도 항상 상수 시간이 소요되므로 1과 0을 개행으로 구분해 출력하면 된다. (수행횟수 1, 최고차항의 차수 0)
