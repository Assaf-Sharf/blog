# ✔️ REGEXP 연산자

%(와일드카드, wildcard) %r% ...

_(언더바) b_____ ==> 5개의 문자가 따라온다 .. 몇개의 문자가 나오는지 알려주는 것
% 자체를 찾고싶을 땐, \%로 구분하는 기호를 사용해 줘야 함 (__\%) 두자리수 퍼센트 찾는 것
