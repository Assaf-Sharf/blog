> ### 톰캣 환경설정

- 시스템 환경 변수 편집
- 변수편집 위쪽 새로만들기
- CATALINA_HOME / 파일 위치 C:\Dev\Program\apache-tomcat-9.0.63
- PATH 파일 편집에서 %CATALINA_HOME%\bin 넣기

---

> ##### 톰캣 사용확인

- cd C:\Dev\Program\apache-tomcat-9.0.63\bin 로 들어가기
- startup.bat 입력  > 새로운 파일창 뜸
- shutdown.bat  입력 > 창 꺼짐 (정상)
- 이상이 발생되면 jdk 환경설정이 잘 되어 있을 수 있다

---

> ##### 이클립스 톰캣 연동

- window - preferences - Server - Runtime Environments - add
- 톰캣 9.0 체크 후 아래 Create a nw local server 체크 - next
- browse - apache-tomcat-9.0.63(bin 폴더 바로 직전 폴더) - jre 선택 - finish
- 이클립스 실행 후 - window - server (없으면 others에서 찾기)
- file - Switch Workspace - 변경 - 재시작
