# SEO

검색 엔진 최적화로 Search Engine Optimization, 줄여서 SEO로 불린다.<br>
검색 엔진 최적화는 검색 엔진으로부터 웹 사이트나 웹 페이지에 대한 웹 사이트 트래픽의 품질과 양을 개선하는 과정이다.<br>
사람들은 보통 검색 결과 사이트의 첫 번째 페이지를 잘 안넘어 가는데, 검색 엔진 최적화를 충족 시키지 못하면 첫 페이지에 노출 될 수 없다.

## 검색 엔진

고객의 검색어(Query)에는 고객이 원하는 요구 사항이 담겨 있고, 검색 엔진은 고객의 **질문 의도에 제일 가까운 답**을 제공해준다.<br>
구글, 네이버, 다음 같은 검색엔진은 고객을 유지하려면 고객의 니즈를 충족시킬 수 있는 답을 제공해 줘야하고, 그렇기 때문에 컨텐츠의 랭킹을 매겨서 상단 노출 시켜준다.<br>
따라서, 검색 엔진에 최적화된 글을 작성해야만 광고 없이 상위에 노출될 수 있다.

<br>

# 왜 검색엔진최적화 인가?

돈을 많이 써서 광고로 상단에 페이지를 띄워도 고객은 광고로 생각하고 신뢰하지 않게 된다.<br>
하지만 검색엔진 최적화를 하면 자연적으로 검색 순위가 높아짐에 따라 Organic Traffic이 증가한다.

![google_search_feature](/images/reference/google_search_feature.png)

통계 자료를 보면 광고 페이지는 클릭율이 현저히 낮으며 또한, 상위 3% 이하로는 클릭율이 낮아지는 것을 확인할 수 있다.<br>
따라서, 상위 3% 안에 나올 수 있게 트래픽을 늘려 결과적으로 매출을 높여야 한다.

## Organic Traffic

Organic Traffic이란 광고나 소셜미디어, 리퍼럴 사이트와 같은 채널을 통해 사이트로 유도되는 트래픽을 제외하고 검색 엔진을 통해 곧바로 유입되거나, 동일한 도메인 안에서 유입되는 트래픽을 말한다.<br>
즉, 네이버, 구글, 다음과 같은 검색엔진의 자연 검색 결과 상위에 노출된 웹페이지 링크를 타고 들어온 웹사이트 방문자 수라고 할 수 있다.

<br>

# 검색 엔진 알고리즘

네이버의 경우 다른 검색엔진과 달리 블로그, 카페, 지식인 등 특수적인 카테고리가 많기 때문에 노출이 쉽지 않다. 즉, 네이버는 하나의 SNS와 비슷하다.<br>
구글의 경우 오직 검색에 집중을 하는 검색엔진이기 때문에 네이버처럼 복잡하지 않다.(전세계 92% 점유율을 차지하고 있다.)<br>

구글은 크롤링(Crawling), 인덱싱(Indexing), 랭킹(Ranking) 3단계 프로세스를 통해 검색 결과 내 가장 적절한 페이지를 상위 노출 시켜준다.

> 참고 : [구글 웹마스터](https://developers.google.com/search/docs/fundamentals/how-search-works?hl=ko)

1. 크롤링은 구글 봇(웹 크롤러)이 웹페이지의 콘텐츠를 복사해서 모든 정보를 수집하고, 수집한 정보를 검색 엔진으로 가져온다.
2. 인덱싱은 구글 봇이 가져온 정보를 주제별로 색인해서 데이터를 보관한다.
3. 랭킹은 색인된 콘텐츠를 검색 의도에 맞춰서 순위를 부여한 다음 사용자가 해당 키워드로 검색했을 때, 랭킹 순서대로 결과를 제공한다.

이 3단계 프로세스를 통해 구글은 서프(SERP, Search Engine Result Page)를 완성한다.

<br>

# SEO로 검색결과 상위 노출

검색 엔진 최적화로 사이트를 상단 노출 시키기 위해서 4단계 과정을 거쳐야한다.

1. 웹 사이트 크롤링을 통해 현재 내 사이트 문제 진단하기
2. 진단 결과를 통해 웹 사이트의 크롤링과 인덱싱 개선하기
3. 검색엔진에 최적화된 사이트로 개선하기
4. Keyword ReSearch를 통해 검색엔진과 사용자를 만족시키는 콘텐츠를 만든다.

<br>

# 구글 SEO로 상위 노출 하는 방법

1) HTTP가 아닌 HTTPS 보안 프로토콜을 사용한다.

2) URL 최적화

서브 도메인보다 서브 폴더 형식을 사용해야 한다.<br>
서브 도메인 형식을 사용하면 검색 엔진은 사이트가 여러 개 있다고 인식하기 때문에, 도메인 점수가 분산된다.<br>
EX) www.kangtaegong.com/blog  www.kangtaegong.com/information 등

3) **robots.txt**를 정리하고 사이트 루트에 위치해둔다.

robots.txt는 사이트맵 위치, 접근 가능한 파일과 그렇지 않은 파일 등의 정보를 알려주는 중요한 역할을 하는 파일이다.<br>
따라서, 필요한 정보가 포함되어있는지 체크 후 `https://www.kangtaegong.com/robots.txt`처럼 사이트 루트에 위치해 둔다.

4) 사이트맵

사이트맵이란 웹사이트 내의 모든 페이지들을 나열한 파일이다.(Ex.책의 목차)<br>
사이트맵 또한 robots.txt와 마찬가지로 루트에 위치하는 것을 권장.

5) Title Tag

검색엔진 최적화를 위해서는 기본적으로 웹 사이트를 HTML 문법에 맞게 작성해야 한다.<br>
Title Tag는 페이지의 제목을 의미하고, 검색 엔진은 Title Tag에 적용된 내용을 검색 결과에 표시한다.<br>
Title Tag는 클릭율에 영향을 미치기 때문에 [구글 키워드 플래너](https://ads.google.com/home/tools/keyword-planner/)나 [구글 트랜드](https://trends.google.com/trends/?geo=KR) 등을 활용하여 키워드 리서치를 최적화 해야한다.
