## 문제  
2751

## solution  
N개의 수가 주어졌을 때, 이를 오름차순으로 정렬하는 프로그램을 작성하시오.
단순한 문제인데 시간초과가 발생했다.

## CODE  

```python
tmp = []
for _ in range(int(input())):
    tmp += [int(input())]
tmp.sort()
for k in tmp:
    print(k)
```
