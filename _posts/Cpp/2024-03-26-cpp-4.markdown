## 1. 반복문의 개요
&nbsp;&nbsp;C++의 반목 구문에는 `while`,`for`,`do-while`이 있다.

### while 반복문
```cpp
while(조건문)
{
  ⋯
}
```
&nbsp;&nbsp;조건이 거짓이면 반복을 종료

#### 센티널 제어 while 반복문
* 센티널(sentinel)이란 보초, 감시병이라는 뜻으로 권한이 없는 사람이 특정한 곳에 들어가지 못하게 하는 존재
* 데이터 처리에서 센티넬은 처리 중지를 나타내기 위해 데이터 리스트에 존재하는 특별한 항목
* 예를 들어, 양의 정수 배열에서 리스트의 마지막 부분에 센티널로 음의 정수를 넣어 리스트의 끝을 나타낸다.
* 이와 같이 끝을 모르는 반복문에서 센티널을 사용하여 반복문을 종료한다.

#### EOF 제어 while 반복문
* `>>` 연산자는 스트립에서 항목을 읽다가 EOF를 만나면 반복문을 종료한다.
* 키보드를 데이터 소스로 할 경우 Ctrl + z 또는 EOF 입력

### for 반복문
```cpp
for (초기화; 조건; 변경)
{
  ⋯
}
```
```cpp
for (초기화; ;변경) {⋯} // 조건 부분이 비어 있으면 true가 입력된 것으로 간주
```
