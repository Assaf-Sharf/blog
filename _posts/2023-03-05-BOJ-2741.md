## 문제  
2741

## solution  
자연수 N이 주어졌을 때, 1부터 N까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.

## CODE  

```python
for k in range(1,int(input())+1):print(k)
```
제출한 코드.  

```python
print(*range(1,int(input())+1))
```
*a를 쓰면 a를 unpacking 한 후에 인자로 넘겨주게 된다.

>a = range(1, 10)
print(a)
# Output: range(1, 10)
print(*a)
# Output: 1 2 3 4 5 6 7 8 9
