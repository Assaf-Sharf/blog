## 문제  
2587

## solution  
숫자 5개를 한줄씩 줄때, 평균과 중앙값 구하기

## CODE  

```python
tmp = []
for _ in range(5):
    tmp += [int(input())]
tmp.sort();print(int(sum(tmp)//5),tmp[2],sep='\n')
```
제출  

```python
print(sum(l:=sorted(map(int,open(0))))//5,l[2])
```
이렇게 할 수도 있다.
