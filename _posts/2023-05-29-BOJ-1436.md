## 문제  
1436  
  
어떤 수에 6이 3개이상 연속으로 들어가는 수를 종말의 수라고 할때, N번째 종말의 수 구하기  

## solution  
str로 바꿔서 666이 있나 체크해보면 될텐데, for문으로 깔끔하게 풀어보려니 잘 생각이 안난다.

## CODE  

```python
i = int(input());n = 0;l = []
while(True):
    n += 1
    if '666' in str(n) : l+=[n]
    if len(l) == i: break
print(l[-1])
```
제출한 코드   

```python
n=int(input())
i=0
while n:i+=1;n-='666'in'%s'%i
print(i)
```
짧은 코드를 보니 while문이 깔끔하게 풀리는게 맞는것 같다.
