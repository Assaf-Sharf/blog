## 혹시... 야근하세요?
남들보다 빠른 작업을 원한다면 업무환경부터 바꿔야한다.
어느 곳에서 일하든 나에게는 업무환경이 무척이나 중요하다.
업무환경은 **효율성**과 직결되기 때문이다.  
![image](https://github.com/user-attachments/assets/ffbdb0a4-6335-46f7-8dea-25cf7a6d7cce)

### 모든 토목 설계는 Auto CAD로 이뤄진다.
Auto CAD는 토목인들의 VS Code와 같다.
이게 없으면 일을 할 수 없다. 소름돋게 비슷한 것은 CAD 마에스트로들이 있다는 것이다.
이들의 지휘봉은 마우스이고, 키보드로는 쉴새없이 명령어를 입력한다.
마우스와 키보드를 동시에 사용해야하는 CAD 작업 특성 상 모든 **명령어는 왼손에서 작업**되고 **오른손으로 마우스만 작업**하게 된다면 업무효율이 거의 2~300%는 올라갔다.  
![image](https://github.com/user-attachments/assets/0609ea27-27b3-4e39-8e2e-6cd006d18824)

### 키보드 위 '마에스트로 셋팅법'
개발을 하면서 우리가 가장 많이 쓰는 VS Code는 다양한 기능의 단축키를 제공한다.
그러나 그 많은 단축키를 외울 필요는 없다.
**Simple is the BEST!**
선택지가 많으면 머리만 아프다. 자주사용하는 것 그것이 업계표준이 아닐까?

**<VS Code 단축키 설정방법>**
먼저 VS Code 왼쪽 하단에 바로 가기 키를 눌러준다.
<img width="1470" alt="image" src="https://github.com/user-attachments/assets/6d6938e8-cb21-4a9f-ad45-c73ec2df5826">

이제 검색창에 아래 검색어를 입력하고 키 바인딩을 바꿔주면 단축키 사용이 가능하다.
확장 프로그램에서 Korean Language Pack을 설치한 유저라면 한글 검색어를, 영문판을 그대로 사용하는 유저라면 괄호()안의 검색어를 검색해주면 된다.

#### 1. 주석생성
- 검색어: 블록 주석 설정/해제 (Toggle Block Comment)
- 단축키: optoin + shift + /
<img width="1168" alt="image" src="https://github.com/user-attachments/assets/4741b03d-4f4d-4143-b8e3-873d5cd259c0">

#### 2. 같은 단어 모두 선택
- 검색어: 일치 항목 찾기의 모든 항목 선택 (Select All Occurrences of Find)
- 단축키: option + shift + a
<img width="1041" alt="image" src="https://github.com/user-attachments/assets/bd1b1aa0-7004-4c24-94c0-aa61e2b18d20">

  
#### 3. 터미널로 focusing
- 검색어: 터미널: 터미널 보기에 포커스 (Terminal: Focus on Terminal View)
- 단축키: ctrl + option + ↓
  <img width="1042" alt="image" src="https://github.com/user-attachments/assets/9d1a6b75-b810-4a61-b44f-650f079c3d9f">

#### 4. editor로 focusing
- 검색어: 보기: 활성 편집기 그룹에 포커스 (Veiw: Focus Active Editor Group)
- 단축키: ctrl + option + ↑
<img width="1053" alt="image" src="https://github.com/user-attachments/assets/4cc10c8e-829b-461e-b36c-4726e7b08b69">

#### 5. 아래 줄에 복사
- 검색어: 아래에 줄 복사 (Copy Line Dwon)
- 단축키: opiont + shift + ↓ 
  <img width="1040" alt="image" src="https://github.com/user-attachments/assets/c8b97fa3-b5b8-4146-a467-d00ac3c41aef">


#### 6. 윗 줄에 복사
- 검색어: 위에 줄 복사 (Copy Line Up)
- 단축키: option + shift + ↑ 
  <img width="1040" alt="image" src="https://github.com/user-attachments/assets/a4f4c801-4bb8-4ea6-abf6-95621b2175fb">

#### 7. 위와 줄 바꿈
- 검색어: 줄 위로 이동 (Move Line Up)
- 단축키: opt + ↑
  <img width="1037" alt="image" src="https://github.com/user-attachments/assets/46ae5fb2-adfa-4fa8-af30-64b6b3434a68">

#### 8. 아래와 줄 바꿈 
- 검색어: 줄 아래로 이동 (Move Line Down)
- 단축키: opt + ↓
<img width="1002" alt="image" src="https://github.com/user-attachments/assets/97fa413e-9af8-44fc-b71e-666bdb8284a5">


#### 9. 라인지우기
- 검색어: 줄 삭제 (Delete Line)
- 단축키: command + d
