- **DOM object 를 어떻게 생성되어있는지 보고 싶다면**

```jsx

[//console.dir](notion://console.dir/)() 안에 넣어주면 "자바스크립트 객체" 로 볼 수 있다.
//DOM 을 이용하여 이벤트 리스너 사용하기
const h2Element = document.querySeclector('h2'); //쿼리 셀렉터는 문자열을 받는다.

//문서에서는 '반갑습니다'인데, '안녕하세요'로 바꾸는 함수를 만들어본다.
//window 에 있는 addEvenetListener 속성 이용한다!

function.replaceH2Tect() {
h2Element.innerText = '안녕하세요';
}
//addEvenetListener 의 규칙
h2Element.addEvenetListener('click', replaceH2Tect)

(첫 번째 인자 : 이벤트 타입)
(두번째 인자 : 첫번째 인자가 트리거가 되어 실행되어질 함수)

- 기전 : 'click'이 라는 이벤트 타입이 h2Element 에서 발생
=> replaceH2Tect 함수가 실행됨.
```
