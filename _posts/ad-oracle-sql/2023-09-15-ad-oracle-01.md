# Relational Databases

Relational Database organizes data into relations or tables of related data. In a relational database, tables are made up of rows, which represent instances of a given entity, and columns, which represent attributes of each entity.

- Relation: A set of columns of related data

> Other types of database
>
> - Graph database: Neo4j
> - Object database: Realm, Objectivity/DB
> - Document database: CouchDB, MongoDB

## Column Data Types

- Number
- Text
- Date
- True/False value
- Binary data
