## Streamlit 이란?

- **python으로 데이터 분석을 위한 웹앱을 쉽게 만들어주는 라이브러리**
  - python 모델을 배포, 그래프 시각화등 다양한 웹어플리케이션을 배포할수 있는 패키지

### DataFrame 출력하기

```python
    'second column': [10, 20, 30, 40]
```

### 차트 및 지도 그리기

- [다양한 reference 링크](https://docs.streamlit.io/library/api-reference#chart-elements)
- 꺽은선 그래프

```python
import streamlit as st
import numpy as np
import pandas as pd

chart_data = pd.DataFrame(
     np.random.randn(20, 3),
     columns=['a', 'b', 'c'])
```

### 레이아웃

```python
import streamlit as st

# 왼쪽 오른쪽 레이아웃 나누기
left_column, right_column = st.columns(2)

# 왼쪽에 버튼 생성
left_column.button('Press me!')

# 오른쪽 배치
with right_column:
    chosen = st.radio(
        '라디오 버튼',
        ("a", "b", "c", "d"))
    st.write(f"선택 : {chosen}")
```

### 진행 상황 표시

```python
import streamlit as st
import time

'진행바'

# 초기설정
latest_iteration = st.empty()
bar = st.progress(0)

for i in range(100):
  latest_iteration.text(f'진행중.. {i+1}')
  bar.progress(i + 1)
  time.sleep(0.1)

'끝!'
```

### 함수 사용하는 방법

- streamlit 에서 함수를 사용할때는 @st.cache

```python
import streamlit as st
import numpy as np
import pandas as pd


@st.cache
def return_df():
    chart_data = pd.DataFrame(
        np.random.randn(20, 3),
        columns=['a', 'b', 'c'])

    return chart_data


data = return_df()

st.dataframe(data)
```
