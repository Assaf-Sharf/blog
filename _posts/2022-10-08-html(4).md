# 이미지 태그

| 속성   | 설명          | 비고                                |
| ------ | ------------- | ----------------------------------- |
| src    | 원본파일 경로 | 절대경로 또는 상대경로              |
| alt    | 대체 텍스트   | 스크린 리더, 원본파일 무효시        |
| title  | 툴팁          | alt의 대체제나 반복이 되어서는 안됨 |
| width  | 너비          | 픽셀 단위의 정수                    |
| height | 높이          | 픽셀 단위의 정수                    |

**태그 작성**

```html
<img src="주소" alt="설명" title="제목" width="너비" height="높이">
```
