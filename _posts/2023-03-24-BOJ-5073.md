## 문제  
5073

## solution  
삼각형의 세 변의 길이가 주어질때 경우에 맞는것 출력

## CODE  

```python
while(True):
    l = list(map(int,input().split()))
    l.sort()
    if l[2] == 0: break
    if l[0] + l[1] <= l[2] or l[0] == 0:
        print('Invalid')
    elif l[0]==l[1] and l[1]==l[2]:
        print('Equilateral')
    elif l[0]==l[1] or l[1]==l[2]:
        print('Isosceles')
    else:
        print('Scalene')
```
