## IPv4 프로토콜
### 역할
네트워크 상에서 데이터를 교환하기 위한 프로토콜이며, **데이터가 정확하게 전달되는 것을 보장하지 않는다.**

### 구조
- Version (4 bit)
- IHL (4 bit)
  - Header Length
- TOS (1 byte)
  - Type of Service
- Total Length (2 byte)
- Identification (2 byte)
- IP Flags (3 bit)
- Fragment Offset (13 bit)
- TTL (1 byte)
  - Time To Live
- Protocol (1 byte)
- Header Checksum (2 byte)
- Source Address (4 byte)
- Destination Address (4 byte)
- IP Option
