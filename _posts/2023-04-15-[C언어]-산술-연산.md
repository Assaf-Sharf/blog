# #C #Arithmetic

C언어에서 산술연산을 공부해볼 차례이다.

```c
#include  <stdio.h>
 int  main() {
	 int a =  1, b =  2;
	   a = a +  1; 
	   printf("a : %d \n", a);
	   b +=  1;
	   printf("b : %d \n", b);
return 0;
}
```

다음과 같이 입력했을 때, `a:2 b:3`이라고 뜬다.

### 1. int
 `int a=1;`에서 int는 정수형 인자를 나타내는 함수이다.
