# #C #Arithmetic

C언어에서 산술연산을 공부해볼 차례이다.

```c
#include  <stdio.h>
 int  main() {
	 int a =  1, b =  2;
	   a = a +  1; 
	   printf("a : %d \n", a);
	   b +=  1;
	   printf("b : %d \n", b);
return 0;
}
```
