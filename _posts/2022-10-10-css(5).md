### list style

* disc, cricle, squre, "-", 이모지, url, lower-alpha, upper-alpha 등
<br>

\* html

```html
<ul>
  <li>ul 아이템 1</li>
  <li>ul 아이템 2</li>
  <li>ul 아이템 3</li>
</ul>
<ol>
  <li>li 아이템 1</li>
  <li>li 아이템 2</li>
  <li>li 아이템 3</li>
</ol>
```


\* css

```css
ul > li{
  list-style: url(https://showcases.yalco.kr/html-css/02-03/yalco.png)
}

ol {
  list-style: lower-alpha;
}
```
