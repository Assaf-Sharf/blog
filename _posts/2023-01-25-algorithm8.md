# 알고리즘 오답노트
## 진료 순서 정하기
### 문제: 외과의사 머쓱이는 응급실에 온 환자의 응급도를 기준으로 진료 순서를 정하려고 합니다. 정수 배열 emergency가 매개변수로 주어질 때 응급도가 높은 순서대로 진료 순서를 정한 배열을 return하도록 solution 함수를 완성해주세요.  
### 예시: 
emergency       	    result   
[3, 76, 24]	            [3, 1, 2]   
[1, 2, 3, 4, 5, 6, 7]	[7, 6, 5, 4, 3, 2, 1]   
[30, 10, 23, 6, 100]	[2, 4, 3, 5, 1]   
```javascript
function solution(emergency) {
    let copy = [...emergency];
    let copy2 = [...emergency];
    let sortArray = copy.sort((a,b)=>b-a);
    for(let i = 0 ; i < sortArray.length ; ++i){
        let findnum = sortArray[i];   
        let index = emergency.indexOf(findnum);
        copy2.splice(index,1,i+1);
    }
    return copy2
}
```
우선 해답을 적었다.  
처음 이 문제를 접근했을때는 sort()를 emergency에 바로 적용하는 바람에 많은 문제가 있었다.  
emergency.sort()를 해버리면 원본 배열도 직접 변경하여 정렬된 배열을 반환한다.  
이를 알아차리는 데에만 꽤나 시간이 걸렸다.  
원본은 그대로 냅두고 emergency 배열을 복사해서 copy에 지정해두고 이를 정렬시켜 문제를 풀었다.
