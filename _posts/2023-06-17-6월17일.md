● for (int i : <range>) {}

이러한 형태의 구문은 범위기반 for루프이며, 이 구문은 컨테이너, 반복가능한 객체의 각 요소를 반복적으로 접근할 때 사용된다.



<range>는 컨테이너, 배열, 초기화 리스트, 문자열 등의 반복가능한 객체이다.



```c++
std::vector<int> numbers = {1, 2, 3, 4, 5};

for (int num : numbers) {
    std::cout << num << " ";
}

// 출력 : 1 2 3 4 5
```





 위의 예시에서 numbers라는 벡터 컨테이너의 각 요소를 num에 할당하여 출력하고 잇다.



● pair<int, int> name



pair 클래스는 <vector>, <utility>, <algorithm> 헤더파일에 존재하는 STL





```c++
pair<type, type> name;
// type에는 타입의 이름, name에는 pair 클래스의 객체의 이름을 지정해주면 된다.

// 이를 응용해 vector와 함께 쓰면
vector<pair<type, type>> name(n);
// 쌍을 원소로 하는 벡터를 만들 수 있다. name은 벡터의 길이
```



```c++
p.first; // 객체의 첫번째 값
p.second; // 객체의 두번째 값
make_pair(value1, value2); // 두 value를 한 쌍으로 만들어줌
p.push_back(make_pair(value1, value2));
// p에 (value1, value2) 대입.
```



● sort()

● <stack>

c에서는 stack을 일일이 구현해야 했지만 c++에는 헤더파일로 있어서 매우 편리했다.

https://www.acmicpc.net/problem/10773

백준 10773번 문제에서 나는 vector를 이용해 push_back(), pop_back()를 사용했는데 다른 풀이들을 보니 stack을 많이 썼다.

```c++
#include <stack>
stack<type> name; // type은 넣고자하는 자료의 타입, name은 stack의 이름을 지정한다.

stack.push(element); // element를 push

stack.pop(); // 맨끝의 요소 pop

stack.top(); // 최상위 데이터 반환

stack.size(); // 스택의 현재 사이즈

stack.empty(); // 스택이 비어있는지 여부

swap(stack1, stack2); // stack1, stack2의 내용을 바꾸고 싶은 경우 swap
```



● c++에서 정수 뒤집기

주어진 정수를 string으로 바꾼 다음 reverse함수를 이용해 뒤집은 뒤 다시 정수로 바꾼다.

```c++
int rev(int n) {
    string s = to_string(n);
    reverse(s.begin(), s.end());
    return atoi(s.c_str());
}
```

atoi = 문자열 -> 정수

atof = 문자열 -> 실수

atol = 문자열 -> long

c_str()은 해당하는 string의 첫번째 문자의 포인터를 반환한다.
