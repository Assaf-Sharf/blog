<p><p>Microsoft is walking back on a controversial feature implemented into one of the latest preview releases of its flagship Windows 10 operating system.</p>
<p>Microsoft had initially implemented a pop-up prompt in its Windows 10 preview release from September 14th. The prompt appeared whenever a user had attempted to install <a href='//alternativeto.net/software/google-chrome/'><img alt='Small Google Chrome icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&height=36&mode=crop&upscale=false' />Google Chrome</a> or <a href='//alternativeto.net/software/firefox/'><img alt='Small Mozilla Firefox icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/firefox_114558.png?width=36&height=36&mode=crop&upscale=false' />Mozilla Firefox</a>. Touting that users already had &quot;the safer, faster browser for Windows 10&quot; in <a href='//alternativeto.net/software/microsoft-edge/'><img alt='Small Microsoft Edge icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/microsoft-edge_85645.png?width=36&height=36&mode=crop&upscale=false' />Microsoft Edge</a>, the pop-up gave users the option to either open up Edge or continue with the installation of their initially intended web browser.</p>
<p>According to <a href="https://www.theverge.com/2018/9/17/17868946/microsoft-windows-10-warning-prompt-chrome-firefox-test" rel="nofollow">Tom Warren at The Verge</a>, Microsoft no longer intends to implement this prompt in next month's aptly titled Windows 10 October 2018 Update, but could incorporate similar pop-up-like prompts in the future.</p>
