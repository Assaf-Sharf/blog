# 재귀함수
- 리스트의 최댓값 찾기
리스트에서 `max()` 함수를 사용하지 않고, 재귀함수를 활용하여 최댓값을 출력하는 문제이다. 
재귀함수를 통해 문제를 해결하는 방법에 대해 알아보도록 하겠다. 

1. 베이스 케이스 작성하기 
- 재귀함수에서 가장 먼저 작성해야 하는 것은 베이스 케이스를 구상하는 것이다. 
리스트에서 가장 큰 값을 찾는 알고리즘으로 가장 마지막에 있는 값과 리스트안의 있는 값을 비교하여 가장 큰 값을 찾는다. 

```Python
def max_list(my_list):
    # 베이스 케이스
    if len(my_list) == 1:
        return my_list[0]
```
`max_list` 라는 함수를 설정한 뒤 `my_list` 라는 리스트를 매개변수로 받는다.  
베이스 케이스의 경우에는 my_list의 길이가 1이 되는 경우가 base case가 된다.  

2. 재귀 케이스 작성하기
- 재귀 케이스에 대해 생각해보도록 하겠다. 재귀케이스는 (n-1)개의 요소들에서 계속해서 최댓값을 찾는 로직을 작성하도록 하겠다

```Python
# 재귀 케이스
max_sublist = max_list(my_list[:-1]) # 첫 (n-1) 개 요소들의 최댓값
if max_sublist >= my_list[-1]:
    return max_sublist
else:
    return my_list[-1]
```
재귀 케이스를 통해 리스트의 (n-1)개의 요소들을 통해 최댓값을 구하게 된다.  
그리고 max_sublist가 my_list의 마지막 요소와의 크기를 비교하여 리스트의 최댓값을 확인하면 된다. 


3. 모범답안
```Python
def max_list(my_list):
    # 베이스 케이스
    if len(my_list) == 1:
        return my_list[0]
    
    # 재귀 케이스
    max_sublist = max_list(my_list[:-1]) # 첫 (n-1) 개 요소들의 최댓값
    if max_sublist >= my_list[-1]:
        return max_sublist
    else:
        return my_list[-1]
```
