<img class="align-center" src="%20http://d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&amp;height=36&amp;mode=crop&amp;upscale=false">
<p><p>Longtime users of the Chrome web browser will have to adapt to the removal of a keyboard shortcut used to quit the application.</p>
<p>In the latest versions of <a href='//alternativeto.net/software/google-chrome/'><img alt='Small Google Chrome icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&height=36&mode=crop&upscale=false' />Google Chrome</a>, a shortcut to quit the web browser, Ctrl-Shift-Q, will now notify users to press Alt-F-X to exit instead. This change comes following <a href="https://bugs.chromium.org/p/chromium/issues/detail?id=243164" rel="nofollow">a discussion on Chromium's official bug tracking page</a> that has been ongoing since May of 2013. Many users mean to press Ctrl-Shift-Tab to change the currently active tab in a browser window. In that same discussion thread for the issue, users are objecting to the recent change.</p>
<p><a href="https://www.ghacks.net/2018/10/22/google-retires-ctrl-shift-q-in-chrome-to-exit-web-browser/" rel="nofollow">As reported by gHacks</a>, the Ctrl-Shift-W keyboard shortcut to close all open tabs was not impacted in any way. Also noted in the gHacks article, good alternatives with the Ctrl-Shift-Q keyboard shortcut still intact include <a href='//alternativeto.net/software/firefox/'><img alt='Small Mozilla Firefox icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/firefox_114558.png?width=36&height=36&mode=crop&upscale=false' />Mozilla Firefox</a> and Firefox-based browsers such as <a href='//alternativeto.net/software/palemoon/'><img alt='Small Pale Moon icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/palemoon_66357.png?width=36&height=36&mode=crop&upscale=false' />Pale Moon</a>.</p>
