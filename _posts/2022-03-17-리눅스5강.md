### 1.사용자 그룹관리

#### 1)사용자,그룹,권한관리

 여러 사람이 사용하는 다중사용자 운영체계인 UNIX와 LINUX는 사용자 묶음의 그룹 개념이 존재한다.

1. 사용자는 여러개의 그룹에 포함될 수 있음
2. 사용자 및 그룹관리 명령어
   - id : 현재 사용자 알아보기
   - groups :  현재 그룹
   - adduser : 사용자 등록
   - addgroup : 그룹을 등록
   - deluser :   유저를 지움
   - delgroup : 그룹을 지움
3. 사용자와 그룹은 시스템 내부에서 숫자로 표시됨
   - uid : 사용자를 표시하는 숫자
   - gid : 그룹을 표시하는 숫자
