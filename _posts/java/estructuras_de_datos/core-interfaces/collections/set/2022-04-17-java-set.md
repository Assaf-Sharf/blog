## Interface - Set

```java
// SuperInterfaces
Collection<E>, Iterable<E>
```

* Colección parecida a la ``interface List``

* Los elementos del conjunto no tienen un orden establecido
  * Ejemplo : **Lista de la compra**

* No se permite elementos duplicados

* Útiles para trabajar con **base de datos** y **queries**

```java
Set<String> conjunto = new HashSet<String>();
```
