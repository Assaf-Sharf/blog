# JVM (Java Virtual Machine)

### JVM

* 자바 프로그램 실행 환경을 만들어주는 소프트웨어
* 자바 코드가 javac.exe에 의해서 .class형식의 바이트 코드로 컴파일되면, 이 코드가 자바 가상 머신에서 실행된다.
* JRE(Java Runtime Environment)에 포함되어 있다.
* OS와 자바 애플리케이션 중간에 존재하고, 자바 애플리케이션이 OS로부터 독립적일 수 있도록 한다.
  * 따라서, 개발자는 애플리케이션 개발에 집중할 수 있음. OS마다 다르게 개발을 진행하지 않아도 된다.
  * OS에 맞는 JVM을 다운받기만 하면된다.(Java는 플랫폼 종속적이나, JVM은 플랫폼 종속적이다.)
