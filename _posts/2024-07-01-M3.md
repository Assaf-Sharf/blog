# 삼각함수, 지수와 로그, 함수의 극한과 연속

## 1. 일반각

일반각은 우리가 흔히 보는 시계처럼, 한 점을 중심으로 두 개의 선이 이루는 각도입니다. 여기서 한 선은 기준이 되고 다른 선은 회전합니다. 

- 예를 들어, ∠$XOP$라는 각을 생각해 봅시다. 
- 이 각의 크기는 $OX$라는 선이 $OP$라는 선으로 회전한 정도로 정의됩니다. 
- 기준 선 $OX$를 '시초선', 회전한 선 $OP$를 '동경'이라고 부릅니다.

각도는 360도를 한 바퀴로 하고, 여러 바퀴를 돌 수 있습니다. 이를 수식으로 표현하면 $360^\circ \times n + \alpha^\circ \ (n \in \mathbb{Z})$처럼 쓸 수 있습니다.

방향을 정할 때, 시계 반대 방향을 '양의 방향', 시계 방향을 '음의 방향'으로 정합니다.

## 2. 호도법

호도법은 원의 반지름과 관련된 각도를 나타내는 방법입니다. 

- 반지름이 $r$인 원에서 길이가 $r$인 호 $AB$를 잡으면, 중심각 ∠$AOB$의 크기를 1 라디안(radian)이라 합니다.
- 라디안을 각도의 단위로 사용하면 $\pi$ 라디안 = $180^\circ$가 됩니다.
- 예를 들어, $1$ 라디안 = $\frac{180^\circ}{\pi}$, $2\pi = 360^\circ$로 각도를 나타낼 수 있습니다.

## 3. 삼각함수

### 3.1 삼각함수의 정의

삼각함수는 원의 중심에서 특정 점을 향하는 선을 이용해 정의됩니다.

![image-20240701135355689](/images/2024-07-01-M3/image-20240701135355689.png)

- 반지름이 $r$인 원에서, 중심 $O$와 점 $P(x, y)$를 잇는 선을 생각해 봅시다. 
- 이때, 각도 $\theta$에 대한 삼각함수는 다음과 같이 정의됩니다:

  - 사인($\sin$): $\sin \theta = \frac{y}{r}$
  - 코사인($\cos$): $\cos \theta = \frac{x}{r}$
  - 탄젠트($\tan$): $\tan \theta = \frac{y}{x}$

### 3.2 삼각함수의 관계

- $\sin$과 $\cos$의 비율로 $\tan$을 구할 수 있습니다: $\frac{\sin \theta}{\cos \theta} = \tan \theta$
- $\sin$의 역수는 코시퀀트($\csc$), $\cos$의 역수는 시퀀트($\sec$), $\tan$의 역수는 코탄젠트($\cot$)입니다.

이 외에도 중요한 관계식이 있습니다:

- $\cos^2 \theta + \sin^2 \theta = 1$

## 4. 삼각함수의 덧셈과 뺄셈 공식

두 각도의 합이나 차의 삼각함수 값은 다음과 같이 구합니다:

- $\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B$
- $\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B$

## 5. 삼각함수의 배각 공식, 반각 공식

- $\cos 2\theta = \cos^2 \theta - \sin^2 \theta$
- $\sin 2\theta = 2\sin \theta \cos \theta$
- $\cos^2 \theta = \frac{1 + \cos 2\theta}{2}$
- $\sin^2 \theta = \frac{1 - \cos 2\theta}{2}$

## 6. 코사인 법칙

코사인 법칙은 삼각형의 세 변과 하나의 각도 사이의 관계를 설명합니다.

- 삼각형 $ABC$에서 각 변을 $a, b, c$라고 하고, $c$에 대하는 각을 $\theta$라 하면,
- 코사인 법칙은 $c^2 = a^2 + b^2 - 2ab \cos \theta$입니다.

## 7. 지수와 로그

### 7.1 거듭제곱

- 실수 $a$와 자연수 $n$에 대해 $a$를 $n$번 곱한 것을 $a^n$이라 합니다.
- 예를 들어, $a^2, a^3$처럼 나타낼 수 있습니다.
- 여기서 $n$을 '지수', $a$를 '밑'이라 부릅니다.

### 7.2 거듭제곱근

- $n$이 2 이상의 자연수일 때, $x^n = a$를 만족하는 수 $x$를 $a$의 $n$제곱근이라 하고, $\sqrt[n]{a}$로 나타냅니다.
- 예를 들어, $\sqrt{a}, \sqrt[3]{a}$처럼 나타낼 수 있습니다.

### 7.3 지수

1) 지수법칙

- $a > 0, b > 0$이고 $x, y$가 실수일 때, 다음과 같은 법칙이 성립합니다:
  - $a^x a^y = a^{x+y}$
  - $\frac{a^x}{a^y} = a^{x-y}$
  - $(a^x)^y = a^{xy}$
  - $(ab)^x = a^x b^x$

2) 0 또는 음의 정수인 지수의 정의

- $a \neq 0$이고 $n$이 양의 정수일 때,
  - $a^0 = 1$
  - $a^{-n} = \frac{1}{a^n}$

3) 유리수인 지수의 정의

- $a > 0$이고 $m, n (n \geq 2)$이 정수일 때,
  - $a^{\frac{m}{n}} = \sqrt[n]{a^m}$, 특히 $a^{\frac{1}{n}} = \sqrt[n]{a}$

4) 무리수인 지수의 정의

- $a > 0$일 때, 무리수 $k$에 대해 $a^k$는 수열의 수렴값으로 정의됩니다.

### 7.4 로그

1) 로그의 정의

- $a > 0, a \neq 1$일 때, $a^x = N$을 만족시키는 실수 $x$를 $\log_a N$이라고 하며, 이를 $a$를 밑으로 하는 $N$의 로그라고 합니다.

2) 로그의 성질

- $\log_a 1 = 0, \log_a a = 1$
- $\log_a (xy) = \log_a x + \log_a y$
- $\log_a \left(\frac{x}{y}\right) = \log_a x - \log_a y$
- $\log_a (x^n) = n \log_a x$ (단, $n$은 실수)

3) 밑의 변환 공식

- $\log_a b = \frac{\log_c b}{\log_c a}$ (단, $c > 0, c \neq 1$)
- $\log_a b = \frac{1}{\log_b a}$ (단, $b \neq 1$)

4) 로그의 여러 가지 성질

- $\log_a^m b^n = \frac{n}{m} \log_a b$ (단, $m \neq 0$)
- $a^{\log_a b} = b$

## 8. 함수의 극한의 정의

### 8.1 함수의 수렴 (convergence)

- 함수 $f(x)$가 $x$의 값이 $a$에 한없이 가까워질 때, $f(x)$의 값이 일정한 값 $L$에 한없이 가까워지면 함수 $f(x)$는 $L$에 수렴한다고 합니다.
- 이를 기호로 나타내면, $\lim_{x \to a} f(x) = L$ 또는 $x \to a$일 때 $f(x) \to L$입니다.

### 8.2 무한대로 가는 함수의 수렴

- $x \to \infty, x \to -\infty$일 때도 비슷하게 정의됩니다.
- 예를 들어, $\lim_{x \to \infty} \frac{1}{x} = 0$처럼요.

### 8.3 함수의 발산 (divergence)

1) 양의 무한대로 발산

- 함수 $f(x)$가 $x$의 값이 $a$에 한없이 가까워질 때, $f(x)$의 값이 한없이 커지면 양의 무한대로 발산한다고 합니다.
- 이를 기호로 나타내면, $\lim_{x \to a} f(x) = \infty$입니다.

예시: $\lim_{x \to 0} \frac{1}{x^2} = \infty$

2) 음의 무한대로 발산

- 함수 $f(x)$가 $x$의 값이 $a$에 한없이 가까워질 때, $f(x)$의 값이 음수이면서 그 절댓값이 한없이 커지면 음의 무한대로 발산한다고 합니다.
- 이를 기호로 나타내면, $\lim_{x \to a} f(x) = -\infty$입니다.

예시: $\lim_{x \to 0} \left(-\frac{1}{x^2}\right) = -\infty$

### 8.4 무한대로 가는 함수의 발산

- $x \to \infty, x \to -\infty$일 때도 비슷하게 정의됩니다.

### 8.5 우극한과 좌극한

1) 우극한 (right limit)

- 함수 $f(x)$가 $x$의 값이 $a$보다 크면서 $a$에 한없이 가까워지는 경우, 이를 $x \to a^+$로 나타냅니다.
- $x \to a^+$일 때, $f(x)$의 값이 일정한 값 $L$에 한없이 가까워지면 $\lim_{x \to a^+} f(x) = L$입니다.

2) 좌극한 (left limit)

- 함수 $f(x)$가 $x$의 값이 $a$보다 작으면서 $a$에 한없이 가까워지는 경우, 이를 $x \to a^-$로 나타냅니다.
- $x \to a^-$일 때, $f(x)$의 값이 일정한 값 $M$에 한없이 가까워지면 $\lim_{x \to a^-} f(x) = M$입니다.

3) 극한값의 존재성

- 함수 $f(x)$에 대해 $x = a$에서의 우극한과 좌극한이 존재하고 그 값이 $L$로 같으면 극한값 $\lim_{x \to a} f(x)$가 존재합니다.
- 이를 $\lim_{x \to a} f(x) = L \iff \lim_{x \to a^+} f(x) = \lim_{x \to a^-} f(x) = L$로 표현합니다.

### 9.2 샌드위치 정리 (Sandwich Theorem)

- 상수 $c$를 포함하는 어떤 열린 구간에 속하는 모든 원소 $x$에 대해 $g(x) \leq f(x) \leq h(x)$이고, $\lim_{x \to c} g(x) = \lim_{x \to c} h(x) = L$일 때, $\lim_{x \to c} f(x) = L$입니다.

## 10. 연속성

### 10.1 함수의 연속의 정의

- 함수 $f(x)$의 정의역의 어떤 구간에 속하는 실수 $c$에 대해, $\lim_{x \to c} f(x) = f(c)$이면, $f$는 $x = c$에서 연속이라고 합니다.

### 10.2 연속성 (Continuity) 판정법

- 함수 $f(x)$는 다음 세 조건을 만족하면 $x = c$에서 연속이고, 그 역도 성립합니다:
  1) $f(c)$가 존재합니다.
  2) $\lim_{x \to c} f(x)$가 존재합니다.
  3) $\lim_{x \to c} f(x) = f(c)$입니다.

## 11. 사잇값의 정리

### 11.1 사잇값의 정리 (Intermediate Value Theorem)

- 함수 $f(x)$가 닫힌구간 $[a, b]$에서 연속이고 $f(a) \neq f(b)$일 때, $f(a)$와 $f(b)$ 사이의 임의의 값 $k$에 대해 $f(c) = k$인 $c$가 열린구간 $(a, b)$에 적어도 하나 존재합니다.

### 11.2 사잇값의 정리의 응용

- 함수 $f(x)$가 닫힌구간 $[a, b]$에서 연속이고 $f(a)$와 $f(b)$의 부호가 서로 다르면, 즉 $f(a)f(b) < 0$이면 방정식 $f(x) = 0$은 열린구간 $(a, b)$에 적어도 하나 존재합니다.

예시: $f(x) = x^2 - 1$에서 $f(0) = -1 < 0, f(2) = 3 > 0$ 에서 열린구간 $(0, 2)$에 방정식 $f(x) = 0$의 근이 적어도 하나 존재함을 알 수 있습니다.

## 12. 극한값 구하는 방법

### 12.1 주어진 점에서 연속일 경우 직접 대입합니다.

예시:

- $\lim_{x \to 1} \frac{2x + 1}{x} = \frac{3}{1} = 3$
- $\lim_{x \to 0} \sec x = 1$

다음 규칙을 이용해 함수가 연속함을 판단합니다:

1) 다항함수: $\mathbb{R}$에서 연속
2) 무리함수: 루트 안이 0 이상인 구간에서 연속
3) 유리함수: 분모가 0이 아닌 구간에서 연속
4) 사인함수, 코사인함수: $\mathbb{R}$에서 연속
5) 탄젠트함수: 점근선을 제외한 구간에서 연속

### 12.2 주어진 점에서 연속이 아닐 경우

1) 인수분해를 이용한 극한값의 계산

- 다항식의 나눗셈꼴인 식은 인수분해를 통해 극한값을 계산할 수 있습니다.

예시: $\lim_{x \to 1} \frac{x - 1}{x^2 - 4x + 3} = \lim_{x \to 1} \frac{x - 1}{(x - 1)(x - 3)} = \lim_{x \to 1} \frac{1}{x - 3} = -\frac{1}{2}$

2) 무리식의 유리화를 이용한 극한값의 계산

- 무리식을 포함한 식은 유리화를 통해 극한값을 계산할 수 있습니다.

예시: $\lim_{x \to 4} \frac{\sqrt{x} - 2}{x - 4} = \lim_{x \to 4} \frac{(\sqrt{x} - 2)(\sqrt{x} + 2)}{(x - 4)(\sqrt{x} + 2)} = \lim_{x \to 4} \frac{x - 4}{(x - 4)(\sqrt{x} + 2)}$
$= \lim_{x \to 4} \frac{1}{\sqrt{x} + 2} = \frac{1}{4}$

3) 다항함수의 나눗셈꼴의 특별한 경우

- $f(x), g(x)$가 다항함수일 때, 다음과 같은 경우에는 분자와 분모의 차수나 최고차항의 계수 비율로 극한값을 구할 수 있습니다:

![image-20240701145010300](/images/2024-07-01-M3/image-20240701145010300.png)

예시:

- $\lim_{x \to \infty} \frac{x^2 - 2x + 1}{2x^2 - x + 4} = \frac{1}{2}$
- $\lim_{x \to \infty} \frac{x^2 - 2x + 1}{x + 1} = \infty$
