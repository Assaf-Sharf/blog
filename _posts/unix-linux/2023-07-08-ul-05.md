# Unix/Linux File Processing

- Files are treated as character sequences
- Directly access each character
- Perform a range of editing tasks

<br>

# Unix/Linux File Type

1. Regular files `-`
   : Text file
   : - Contain printable ASCII characters
   : - Sometimes called regular/ordinary/ASCII file
   : - Ex) documents, source code, etc
   : Binary file
   : - Contain non-printable characters
   : - Ex) machine language code
2. Directories `d`
3. Special file
   : Character special file `c`, block special file `b`
