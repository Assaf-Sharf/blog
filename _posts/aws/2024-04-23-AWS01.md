## AWS란?

AWS(Amazon Web Services)는 아마존에서 제공하는 클라우드 컴퓨팅 플랫폼입니다. 2006년에 최초 출시된 AWS는 현재 전 세계에서 가장 폭넓게 사용되는 클라우드 서비스 중 하나입니다.

AWS는 `IaaS(Infrastructure as a Service)`, `PaaS(Platform as a Service)`, `SaaS(Software as a Service)`를 모두 제공합니다. 

개발자와 기업이 가상 컴퓨팅 리소스에 쉽게 액세스할 수 있도록 지원하여 빠르고 유연한 리소스 프로비저닝이 가능합니다.

### AWS의 주요 특징

1. **광범위한 글로벌 서비스**: AWS는 전 세계 24개 지역의 77개 가용 영역에서 200가지 이상의 클라우드 서비스를 제공합니다.

2. **손쉬운 확장성**: 필요에 따라 리소스를 쉽게 확장하거나 축소할 수 있으므로 비용 효율성이 높습니다.

3. **보안 및 규정 준수**: AWS는 강력한 보안 기능과 업계 인증을 갖추고 있습니다. (SOC, PCI-DSS, HIPAA 등)

4. **안정적인 인프라**: AWS는 99.99% 이상의 가용성으로 매우 안정적으로 운영됩니다. 

5. **비용 효율성**: 필요한 만큼만 리소스를 사용하고 비용을 지불하므로 비용을 최적화할 수 있습니다.

6. **통합 서비스**: 다양한 AWS 서비스들이 원활하게 통합되어 애플리케이션 배포와 운영이 간편합니다.

### AWS 주요 서비스

AWS의 200여 개 서비스 중 가장 잘 알려진 서비스는 다음과 같습니다

![AWS 주요 서비스](https://velog.velcdn.com/images/mino_dev/post/ed8ebbf2-054d-4c93-849e-88676a094275/image.png)

| 서비스 | 설명 |
|-----------|----------|
| EC2 | 가상 컴퓨팅 환경을 제공하는 IaaS 서비스입니다. |
| S3 | 객체 스토리지 서비스로 안전하고 내구성 있는 데이터 저장소를 제공합니다. |  
| RDS | 관계형 데이터베이스 서비스로 MySQL, PostgreSQL, Oracle, SQL Server 등을 지원합니다. |
| VPC | 가상 프라이빗 클라우드를 구성하여 리소스를 격리하고 보안을 강화할 수 있습니다. |
| Lambda | 서버리스 아키텍처에서 코드를 실행할 수 있는 이벤트 기반 컴퓨팅 서비스입니다. |
| CloudFront | CDN 서비스로 웹 콘텐츠를 전 세계에 빠르게 전송합니다. |

이외에도 IoT, 머신러닝, 분석, 개발자 도구, 보안, 엔터프라이즈 애플리케이션, 모니터링 등 다양한 카테고리에서 AWS는 혁신적인 솔루션을 제공하고 있습니다.

### AWS 요금 및 지원 모델  

AWS 요금 체계는 서비스 유형과 사용량에 따라 다릅니다. 대부분 서비스는 사용한 만큼 과금되는 **종량제(Pay-as-you-go)** 모델을 따릅니다. 장기 사용 시 할인 혜택을 주는 **예약 인스턴스**와 초과 사용분에 대해 저렴한 요금을 부과하는 **스팟 인스턴스** 등의 옵션도 있습니다.

AWS는 프리 티어로 1년간 일부 서비스를 무료로 사용해볼 수 있습니다. 또한 기술 지원 플랜에 가입하면 AWS 전문가로부터 기술 자문을 받을 수 있습니다.

AWS는 클라우드 컴퓨팅의 발전을 선도하며 기업, 개발자, 스타트업 등에게 유연하고 안전하며 비용 효율적인 인프라를 제공합니다. 앞으로 AWS 관련 실무 경험과 지식을 쌓아가며 더욱 깊이 있게 다뤄보겠습니다.
