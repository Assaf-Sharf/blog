# md 작성 가이드

기본적인 md 문법이고 plugin 중에 Makrdown Shortcuts 도 다운 받아서 적절히 사용하면 좋음

#### Syntax
  <pre>
  <code>
    Header
    ===

    ---
    </code></Pre>

### 9. Add Img
plug-in 중에 Paste Image 를 설치 후 경로를 설정 해줘야함 
설정 -> 작업 영억 -> Path Image : Path 부분을
../images 로 변경

원하는 이미지 복사 후 Ctrl + Alt + V 하면 아래 코드가 생성 됨

```
1. 복사한 크기만큼 이미지를 붙여 넣는 방법
![](../images/2022-04-09-15-08-51.png)
2. 크기를 주는 방법 (html 의 img 태그를 이용)
<img src="../images/2022-04-09-15-09-42.png" width="100">
```
#### example 
1. ![](../images/2022-04-09-15-08-51.png)<br><br>
2. <img src="../images/2022-04-09-15-09-42.png" width="200">
