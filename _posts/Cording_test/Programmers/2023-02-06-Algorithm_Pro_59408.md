### 코드

##### ver(1)

```sql
SELECT COUNT(DISTINCT NAME) as count 
  FROM ANIMAL_INS;
```


### 풀이

##### DISTINCT 중복제거

1. 기본적인 형태

```sql
SELECT DISTINCT 컬럼명 
FROM 테이블
```

2.  DISTINCT  뒤에 2개 이상 컬럼 사용

```sql
---테이블에서 컬럼명1, 컬럼명2 합쳐서 중복인 행 제거
SELECT DISTINCT 컬럼명1, 컬럼명2
FROM 테이블
```
