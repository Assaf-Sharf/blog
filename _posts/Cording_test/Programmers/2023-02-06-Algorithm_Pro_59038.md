### 코드

##### ver(1)

```sql
SELECT DATETIME as 시간
  FROM ANIMAL_INS
ORDER BY DATETIME
 limit 1;
```
##### ver(2)

```sql
SELECT MIN(DATETIME) AS 시간 
  FROM ANIMAL_INS;
```

### 풀이

##### LIMIT

 - LIMIT는 결과에서 몇 개의 행을 반환할 것인지 출력을 제한하는 구문이다.

```sql
SELECT 컬럼명
 FROM 테이블
LIMIT 개수;
```

##### Offset 지정


```sql
SELECT 컬럼명
 FROM 테이블
LIMIT offset, 개수;
```
