### 코드

##### ver(1)

```sql
SELECT NAME, COUNT(NAME) as COUNT
FROM ANIMAL_INS
GROUP BY NAME
HAVING COUNT(NAME) >= 2
ORDER BY NAME ASC
```

##### ver(2) - 서브쿼리 이용

```sql
SELECT NAME, COUNT
FROM (
    SELECT NAME, COUNT(NAME) AS COUNT
    FROM ANIMAL_INS
    GROUP BY NAME
    ) SQ1
WHERE COUNT >= 2
```

### 풀이

##### GROUP BY 절

 데이터들을 원하는 그룹으로 나눌 수 있다.

- 컬럼 그룹화

```sql
SELECT 컬럼 FROM 테이블 WHERE 조건식 GROUP BY 그룹화할 컬럼;
```

- 컬럼 그룹화 후에 조건 처리

```sql
SELECT 컬럼 FROM 테이블 GROUP BY 그룹화할 컬럼 HAVING 조건식;
```
