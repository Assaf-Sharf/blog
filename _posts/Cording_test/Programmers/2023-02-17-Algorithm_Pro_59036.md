### 코드

##### ver(1) - IN() 사용

```sql
SELECT ANIMAL_ID, NAME, SEX_UPON_INTAKE
FROM ANIMAL_INS
WHERE NAME IN ('Lucy','Ella','Pickle','Rogan','Sabrina','Mitty')
ORDER BY ANIMAL_ID;
```

---

### 풀이

##### IN()


IN 연산자를 사용하면 WHERE절에 여러 값을 지정 할 수 있다.  **OR** 연산과 유사하다.

```sql
WHERE 컬럼명 IN ('값', '값')
```
