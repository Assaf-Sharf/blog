### 코드

##### ver(1) - MAX 사용

```sql
SELECT *
  FROM FOOD_PRODUCT
 WHERE PRICE = (SELECT MAX(PRICE) FROM FOOD_PRODUCT);
```

##### ver(2) - ORDER BY, LIMIT 사용

```sql
SELECT *
  FROM FOOD_PRODUCT
 ORDER BY PRICE DESC
 LIMIT 1;
```


---

### 풀이

##### MAX, MIN 함수

가장 큰 값과 작은 값을 찾고 싶을 때 사용한다.

- 예시)

```sql
SELECT MAX(컬럼) FROM 테이블;
```

```sql
SELECT MIN(컬럼) FROM 테이블;
```
