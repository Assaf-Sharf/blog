### 문제

RGB거리에는 집이 N개 있다. 거리는 선분으로 나타낼 수 있고, 1번 집부터 N번 집이 순서대로 있다.

집은 빨강, 초록, 파랑 중 하나의 색으로 칠해야 한다. 각각의 집을 빨강, 초록, 파랑으로 칠하는 비용이 주어졌을 때, 아래 규칙을 만족하면서 모든 집을 칠하는 비용의 최솟값을 구해보자.

- 1번 집의 색은 2번 집의 색과 같지 않아야 한다.
- N번 집의 색은 N-1번 집의 색과 같지 않아야 한다.
- i(2 ≤ i ≤ N-1)번 집의 색은 i-1번, i+1번 집의 색과 같지 않아야 한다.

### 입력

첫째 줄에 집의 수 N(2 ≤ N ≤ 1,000)이 주어진다. 둘째 줄부터 N개의 줄에는 각 집을 빨강, 초록, 파랑으로 칠하는 비용이 1번 집부터 한 줄에 하나씩 주어진다. 집을 칠하는 비용은 1,000보다 작거나 같은 자연수이다.

### 출력

첫째 줄에 모든 집을 칠하는 비용의 최솟값을 출력한다.

### 풀이

먼저, 첫 번째 집을 칠하는데 드는 비용을 rgb 리스트에 입력 받는다.
두 번째 집부터 각 색깔별로 칠하는 경우의 최소 비용을 계산해나간다.
예를 들어, 현재 집을 빨강으로 칠할 때, 이전 집을 초록으로 칠한 경우와 파랑으로 칠한 경우 중에서 최소 비용을 계산해야 한다.
이전 집을 칠할 때의 최소 비용은 이미 rgb 리스트에 저장되어 있으므로, 이를 이용하여 현재 집을 칠할 때 드는 비용을 계산한다.
모든 집을 칠할 때의 비용이 저장된 각각의 경우 중에서 최솟값을 출력한다.

### 코드

#### ver(1) 

```python
import sys

# 입력값 받기
input_str = sys.stdin.readline
n = int(input_str())
rgb = []
for i in range(n):
    a, b, c = map(int, input_str().split())
    rgb.append([a, b, c])

# 두 번째 집부터 각 색깔별로 칠할 비용 계산하기
for i in range(1, n):
    # 현재 집을 빨강으로 칠하는 경우 최소 비용
    rgb[i][0] += min(rgb[i-1][1], rgb[i-1][2])
    # 현재 집을 초록으로 칠하는 경우 최소 비용
    rgb[i][1] += min(rgb[i-1][0], rgb[i-1][2])
    # 현재 집을 파랑으로 칠하는 경우 최소 비용
    rgb[i][2] += min(rgb[i-1][1], rgb[i-1][0])

# 마지막 집을 칠하는 각각의 색깔 중에서 최소 비용 출력하기
print(min(rgb[n-1]))
```
