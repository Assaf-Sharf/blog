<h5>저장되어 있는 데이터에서 조건으로 검색해보기[find-전체 정보 출력(list)]<br>
    ※ findOne : 매칭되는 여러개의 document 중 맨 첫번째 document만 선택 <br>

  - find() 괄호안에 인자가 없기에 모든 document를 가져온다.
    - db.people.find()<br><br> 
    
  - 전체 document를 가져오는데 항목은 user_id와 status만 추출
    - db.people.find({ }, { user_id: 1, status: 1 })<br><br>
    
  - 자동생성되는 _id를 빼고자 할 때는 _id:0으로 입력하면 된다.
    - db.people.find({ },{ user_id: 1, status: 1, _id: 0 })<br><br>

  - status가 A인 document만 추출
    - db.people.find({ status: "A" })<br><br>

  - status가 A이고 age가 50인(and) document만 추출
    - db.people.find({ status: "A", age: 50 })<br><br>

  - status가 A이거나 age가 50인(or) document만 추출($or로 표기)
    - db.people.find({ $or: [ { status: "A" } , { age: 50 } ] })<br><br>

<h5> >,<, =, != 조건 사용해보기 </h5>

  - $eq     =   equal
  - $gt     >   greater than
  - $gte    >=  greater than or equal
  - $in         in an array.
  - $lt     <   less than
  - $lte    <=  less than or equal
  - $ne     !=  not equal
  - $nin        none of the values specified in an array.<br><br>

<h5> 다양한 검색 예제 </h5>

  - db.people.find({ age: { $gt: 25, $lte: 50 } }) 
  - db.people.find( { age: { $nin: [ 5, 15 ] } } ) 
  - db.people.find( { user_id: /bc/ } )
  - db.people.find( { user_id: { $regex: /bc/ } } )   어디든 bc가 포함된 document
  - db.people.find( { user_id: /^bc/ } )
  - db.people.find( { user_id: { $regex: /^bc/ } } )  오직 bc로 시작하는 document
  - db.people.find( { status: "A" } ).sort( { user_id: 1 } )   오름차순
  - db.people.find( { status: "A" } ).sort( { user_id: -1 } )  내림차순
  - db.people.count()
  - db.people.find().count()  
  - db.people.count( { user_id: { $exists: true } } )
  - db.people.find( { user_id: { $exists: true } } ).count()
  - db.people.count( { age: { $gt: 30 } } )
  - db.people.find( { age: { $gt: 30 } } ).count()
  - db.people.distinct( "status" ) 
  - db.people.findOne()
  - db.people.find().limit(1)<br><br>

<h5> 특정 조건에 해당하는 데이터를 변경하기</h5>

  -  { $set: {  } } : field값 설정,   { $inc: { } } : 괄호안의 값만큼 field 값을 증감
  -  updateOne : 매칭되는 여러개의 document 中 첫번째 document를 업데이트
  -  updateMany : 매칭되는 list of document(전체)를 업데이트<br>
<h5> Document 삭제 </h5>
  -  removeOne : 매칭되는 여러개의 document 中 첫번째 document를 삭제
  -  removeMany : 매칭되는 list of document(전체)를 삭제<br><br>
