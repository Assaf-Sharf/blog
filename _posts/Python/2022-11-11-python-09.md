# ✔️ Downcast 필요성

컴퓨터가 프로그램을 실행하는 과정을 간단히 해보면 아래 네단계를 거친다.
1. CPU가 코드 실행에 필요한 데이터들을 하드디스크에 가져옴
2. 가져온 데이터를 RAM 메모리에 올림
3. CPU가 연산작업을 하여 결과물을 받아옴
4. 사용자에게 결과물을 반환

빅데이터를 처리하기 위해선 RAM 사용량이 매우매우 큰데, 실제로 사용할 수 있는 RAM 크기는 이미 정해져있다! (나의 컴퓨터 사양은 이미..💻)
