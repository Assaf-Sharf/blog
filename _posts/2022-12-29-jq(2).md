```html
<div class="a"></div>
```

```css
div.a{
  width: 50%;
  height: 500px;
  background-color: red;
  border: 100px solid pink;
  
}
 
div.a.b{
  background-color:green;
}
```

```js
let $div = $('div')
 
 
$div.click(function(){
  $(this).toggleClass('b')
 
// if ( $(this).hasClass('b')){
//   $(this).removeClass('b');
// }
// else {
//  $(this).addClass('b');
//  }
});

```

	• ToggleClass를 이용하면 class를 추가했다 삭제했다를 할수 있음
	• ToggleClass가 없었다면 주석에 있는 부분으로 했어야함
