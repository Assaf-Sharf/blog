# Javascript 함수 정의

## 1. 일반함수
### 예시

```js
function add(x,y){
    return x+y;
}
add(1,3) // 출력 : 4
```

## 2. 익명함수
### 형태

```js
function(){
    console.log("안녕하세요");
}
```

익명함수는 리터럴방식으로 사용되고, 변수에 저장
### 예시

```js
let add = function(x,y){
    return x+y;
}
add(1,3) // 출력 : 4
```
