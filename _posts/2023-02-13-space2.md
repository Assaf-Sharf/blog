> 백준 2739  /  구구단

```python
num = int(input())

for i in range(9):
    print(str(num) + " * " + str(i+1) + " = " + str(num*(i+1)))
```

> 백준 10950  /  a+b 출력

```python
T = int(input())

for i in range(T):
    A,B = map(int, input().split())
    print (A+B)
```

> 백준 8393  /  1부터 N까지

```python
n = int(input())
sum = 0

for i in range(n):
    sum += (i+1)
print(sum)
```
