## 문제  
2908

## solution  
숫자 두개를 입력하면 숫자를 둘다 거꾸로 읽어서 큰 수를 출력하는 문제.

## CODE  

```python
def reverse_num(str_num):
    result = ''
    for reverse_idx in range(len(str_num)-1,-1,-1):
        result += str_num[reverse_idx]
    return int(result)

def main():
    tmp = input().split()
    print(max(reverse_num(tmp[0]),reverse_num(tmp[1])))
    return None

main()
```
