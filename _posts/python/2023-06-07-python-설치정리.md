# 설치 리스트 확인(pip list)
```bash
pip list
```

# 설치 리스트 메모장 저장 (pip freeze)
``` bash
pip freeze > requirements.txt
```

# 설치 리스트 저장 (pip install -r)

```bash
pip install -r requirements.txt
```
