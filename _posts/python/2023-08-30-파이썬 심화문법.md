# 심화문법정리

## 1. 자료형심화

### 1-1. 문자열메서드

- 소문자를 대문자로 바꾸는 방법

```
"write text".upper()
```

- 대문자를 소문자로 바꾸는 방법

```
"write text".lower()
```

- 문자열 바꾸기

```
"write text".replace("text","image")
```

- 문자열 위치 찾기

찾고자 하는 문자열의 첫인덱스 위치를 return 해준다.

```
"hello World".find("World")
```

- 문자열 개수 세는법

```
"hello world is a part of the virtual world".count("world")
```

- 문자열 분리하는법

```
"name:number:height".split(":")
["name","number","height"]
```

- 문자열 연결하는 법

```
':'.join(["name","number","height"])
"name:number:height"
```

- 공백삭제하는 법
|메서드|결과값|
|---|---|
|lstrip()|왼쪽 공백만 삭제|
|rstrip()|오른쪽 공백만 삭제|
|strip()|양쪽 공백모두 삭제|

### 1-3. 리스트다루기

- 리스트에 데이터 추가

```
name.append('jspark')
```

```
name.count('jspark')
```

- 리스트 정렬

### 1-4. 리스트 내포

- comprehension?
for문, if문등 지정하여 리스트를 간편하게 만드는것.

- for문 포함한 리스트 내포
표현식 for 변수 in 순회가능한 데이터

```
nums = [i for i in range(5)]
[0,1,2,3,4]
```

- if문 포함한 리스트 내포
표현식 for 변수 in 순회가능한 데이터 if 조건식

```
nums = [i for i in range(10) if i%2==0]
[0,2,4,6,8]
```

- if else 사용시엔 for문과 if문 순서바꿔서 리스트 내포해라

``` 
result = []
for item in items:
    if item != None:
        result.append(item)
    else:
        result.append('')
print(result)

# 리스트 내포 사용 후
result = [i if i != None else '' for i in items]
```

## 2. 함수

### 2-1. 다양한 매개변수

- 기본 매개변수(default parameter)
함수 정의할 때 매개변수의 기본 값을 지정가능

```
def post_info(title, content='내용없음')
  print('title:',title)
  print('content:',content)

>>> post_info('attend!!')

title:attend!!
content:내용없음
```

- 키워드 매개변수(keyword parameter)
매개변수의 순서 지키지 않아도 된다

```
def post_info(title, content)
  print('title:',title)
  print('content:',content)

>>> post_info(content="no!!", title='why not!!')

title:why not!!
content:no!!
```

- 위치 가변 매개변수(*args)
매개변수 앞에 *가 붙음 (튜플형)
가변 매개변수 = 개수가 정해지지 않은 매개변수

```
def print_fruits(*args):
  for arg in args:
    print(arg)

>>> print_fruits('apple','banana','melon')
apple

banana

melon
```

- 키워드 가변 매개변수(**kwargs)
매개변수 앞에 **가 붙는다(딕셔너리형)

```
def post_info(**kwargs):
  for key, value in kwargs.items():
    print(f'{key}:{value}')

>>> post_info(title='why not!!', content="no!!" )

title:why not!!
content:no!!
```

### 2-2. 람다(lambda) 함수

- 람다함수란?
이름을 지을 필요도 없을 간단한 형태의 함수
다른 함수의 인자(argument)로 넣을 수 있다.
코드가 간결해 지고, 메모리가 절약된다.

- 람다 함수 사용방법

~~~
(기존함수 정의방법)
def 함수이름(매개변수):
  return 결과
def minus_one(a):
  return a-1

>>> minus_one(10)
9

(람다함수 정의방법)
lambda 매개변수 : 결과

lambda a:a-1

>>> (lambda a:a-1)(10)
9

~~~

- 람다 함수 사용방법(if 문 사용)

~~~
(기존함수 정의방법)
def is_positive_number(a):
  if a>0:
    return True
  else:
    return False

>>> is_positive_number(-1)
False

(람다함수 정의방법)
lambda a : True if a > 0 else False

>>> lambda a: True if a > else False(-1)
False
~~~
