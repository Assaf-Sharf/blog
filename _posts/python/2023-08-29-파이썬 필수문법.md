## 필수문법정리

1. 자료형과 변수
    1-1. 숫자형,문자열, 불린형
        - 숫자형 : 숫자, 정수, 실수
        - 문자열 : 문자를 나열한것
        - 불린형 : 참 또는 거짓
    1-2. 변수
        - 변수 : 데이터를 저장할 공간
2. 연산
    2-1. 연산자
        - 연산 : 수나 식을 일정한 규칙에 따라 계산하는것
        - 대입/산술/비교/논리연산
        - 산술연산 
        |산술기호|의미|
        |---|---|
        |/|나누기|
        |//|몫|
        |%|나머지|
        |**|제곱|
        - 논리연산
        |논리연산자|의미|
        |---|---|
        |A and B |A,B 모두 참이라면 True|
        |A or B|A,B 중 하나라도 참이라면 True|
        |not A|A가 참이라면 False|
        - 멤버십 연산
        |멤버십연산자|의미|
        |---|---|
        |in|포함되어 있다|
        |not in|포함되어 있지 않다|        
    2-2. 입력과 자료형 변환
        ```
        x=input("입력하세요>>")
        ```

6. 클래스
    6-1. 클래스와 객체
        - 클래스라는 틀을 이용하여 객체라는 붕어빵을 만든다
        ![image](https://github.com/OC-JSPark/oc-jspark.github.io/assets/46878973/a7cbaeea-d470-446e-b5e1-1f0e2edf22a5)
        - 속성 : 클래스를 나타내는 특징
        - 메서드 : 동작을 나타낸다.
        ```
        class 클래스이름:
          def 메서드 이름(self):
            명령블록
        ```
        - 호출하기
        ```
        인스턴스 = 클래스이름()
        인스턴스.메서드()
        ```
        - 인스턴스는 클래스가 코드상에 존재할수 있게하여 사용가능 하도록 할때 사용한다.
        - 파이썬에서는 자료형도 클래스이다.
