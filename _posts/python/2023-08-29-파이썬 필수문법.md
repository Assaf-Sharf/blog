### 2-1. 연산자

    - 연산 : 수나 식을 일정한 규칙에 따라 계산하는것

|논리연산자|의미|

|A and B |A,B 모두 참이라면 True|

|not A|A가 참이라면 False|

|멤버십연산자|의미|

## 5. 튜플과 딕셔너리

### 5-1. 튜플의 특징

    - 시퀀스 자료형

    - 수정,추가,삭제 불가능한 리스트

    - 메모리 사용이 효율적

    - 읽기만 가능하기 때문에 데이터 손실염려 없음.

### 5-2. 튜플 만들기

    - 예시
```
a=("나는", 36, True)
a=(30,)
```

### 5-3. 튜플 to list로 변환

```
a = tuple([5,6,7])

x=list(range(10))
a=tuple(x)

x=5,6,7
a=list(x)
```

### 5-4. 패킹과 언패킹

    - 패킹 : 여러개의 데이터를 하나의 변수에 할당

```
numbers = 3,4,5
```

    - 언패킹 : 컬렉션의 각 데이터를 각각의 변수에 할당

```
a,b,c = numbers
```

### 5-5. 튜플함수

    - 특정값의 인덱스 구하기
```
a.index(20)
```

    - 특정값의 개수
```
a.count(30)
```

    - 최대값, 최소값

```
max(a), min(a)
```

    - 합계

```
sum(a)
```
