### 5-2. 튜플 만들기

## 7. 클래스

### 7-1. 클래스와 객체

    - 클래스라는 틀을 이용하여 객체라는 붕어빵을 만든다

![image](https://github.com/OC-JSPark/oc-jspark.github.io/assets/46878973/e0ddb1ef-45ac-4d89-a174-d88ae5e6e405)


    - 속성 : 클래스를 나타내는 특징

    - 메서드 : 동작을 나타낸다.

~~~
class 클래스이름:
    def 메서드 이름(self):
    명령블록
~~~

    - 호출하기

~~~
인스턴스 = 클래스이름()

~~~

    - 인스턴스는 클래스가 코드상에 존재할수 있게하여 사용가능 하도록 할때 사용한다.

    - 파이썬에서는 자료형도 클래스이다.

### 7-2. 생성자

    - 생성자(__init__)함수
      
    init 생성자는 인스턴스 만들때 반드시 호출되는 메서드이다
    인스턴스=클래스() : 이렇게 가로 닫힐때 init이 활성화되어 호출이 될 수 있다.
    생성자 : 인스턴스를 만들때 생성되는 메서드를 지칭한다.

    객체는 인스턴스를 포함한다.

    - self 란?

    __init__(self) : self는 인스턴스의 자기자신을 가르키게 된다.
    self.name = name : 현재 호출되어진 인스턴스 자리에 name변수에다가 name으로 대체하라는 뜻이다.

    - 속성추가하기

~~~
(속성추가1)

(속성추가2)

    def __init__(self, play, speed, health):
        self.play = play
        self.speed = speed
        self.health = health
~~~

### 7-3. 상속

![image](https://github.com/OC-JSPark/oc-jspark.github.io/assets/46878973/2b4c59ec-0229-4e1f-bf95-894a635cf6f2)

상속사용방법

부모클래스에는 반드시 생성자가 있다.
자식 클래스에는 생성자 생략가능

~~~
(부모클래스 정의)
class sports:
    def __init__(self, name, health, attack):
        self.name = name
        self.health = health
        self.attack = attack
    def move(self):
        print("전방으로 달리기")

(자식클래스 정의)
class soccer(sports):
    pass

class swim(sports):
    def move(self):
        print("숨쉬기")

class running(sports):
    def move(self):
        print("달리기")

~~~
