# 정의
plugin 이란 응용 프로그램에 특정 기능을 추가하는 소프트웨어 구성 요소이다.  

library는 응용 프로그램의 개발 프로세스 과정에서 컴퓨터 프로그램에서 사용하는 비 휘발성 리소스 모음이다.  
특징으로는, 기능을 사용하기 위해서는 라이브러리 내부를 몰라도 사용가능하다는 점이 있다.  

# 용법

또한 플러그인은 새로운 기능을 추가하고 응용 프로그램을 줄이며 타사 개발자가 응용 프로그램을 확장 할 수 있도록 도와줍니다.


 반대로 라이브러리는 소프트웨어 응용 프로그램을 효율적으로 개발하고 코드 재사용 성을 개선하는 데 도움이됩니다. 

# 결론

플러그인과 라이브러리의 기본적인 차이점은 **플러그인은 애플리케이션의 기능을 향상시키는 확장**이며, 라이브러리는 **소프트웨어를 개발하는 데 도움**이되는 클래스와 함수의 집합이라는 점입니다.
