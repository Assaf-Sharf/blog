<h6>class-card를 6개로 만들면 일부 우측의 카드는 화면을 넘치게 된다.<br>이를 해결하기 위해 inner에 overflow 옵션을 hidden으로 설정하면 화면에서 넘쳐나지 않게 된다. 다만 hidden 옵션은 padding값은 적용을 하지 않는다.   </h6>

```css
  .inner {
    width: 100%;
    height: 100%;
    margin: 0 auto;
    padding: 0rem 2rem;
    overflow: hidden;
  }
```
<br>
<h6>그래서 class-list 內 overflow값도 hidden으로 설정하면 inner의 width값에 맞게 짤린다.  </h6>

```css
  .class-list {
  display: flex;
  list-style: none;
  padding: 0;
  overflow: hidden; /* overflow 를 hidden 으로 설정하면, transform 시 이미지 짤림 */
}


```
