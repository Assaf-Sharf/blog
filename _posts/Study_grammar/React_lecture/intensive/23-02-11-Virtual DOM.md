# DOM이란?

Document Object Model

웹페이지는 수많은 컴포넌트로 구성(nav 요소들이 대표적)

우선 태그하나하나는 element

DOM == element를 트리형태로 표현한 것
DOM에서 element요소들을 'Node'라고 부름
각 노드는 Api를 제공 (Api : HTML요소에 접근해 수정가능한 함수정도로 이해)

form validation도 접근하여 가져와서 확인
