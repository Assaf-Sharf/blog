# Payload

: 탑재화물 즉, 어떠한 실체를 전달하는 역할



## 개념

: state와 action이 들어왔을 때 payload값이 있다면,

action 객체라는를 보낸다는 것은
action type을 payload만큼 처리해서 보내라는 의미

Ex)

action type == state+1이고 payload가 3이면?
+3만큼을 하라는 의미
