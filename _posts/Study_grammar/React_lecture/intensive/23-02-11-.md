# Redux Start

Redux: 중앙 state 관리소를 사용할 수 있게 도와주는 패키지(라이브러리).

+ yarn or npm으로 설치

## redux가 생긴이유

1. ### props driling이 일어날 경우 불필요한 작업이 필요

   고로, 중앙데이터 관리소를 만들어 이 데이터들을 각 컴포에 전송



2. ### Context API  VS  Redux

   컨텍스트는 provider값이 바뀔경우 이 컨텍스트를 참조하는 모든 컴포넌트들이 

   리렌더링되었기 때문에 상태관리툴로서의 역할로는 부족
   ==> 리덕스는 이 비효율적인 부분들을 보완



일반 state==Local state
전역 state==Global state 

리덕스는 이 Global  state(중앙 데이터 관리소)라고 생각하면됨 



Global 스테이트는 불필요한 state들은 참조할 필요가 없이, 
필요한 스테이트만 참조하여 데이터를 관리할 수 있음
