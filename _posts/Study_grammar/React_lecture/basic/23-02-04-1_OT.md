> ## SPA?

: Single Page Application 한개의 페이지로 이루어짐 != MPA(multi page app)

멀티페이지는 어떤 기능을 수행하면 사이트 다시 로딩...
SPA는 한개의 페이지에서 해당 기능을 살짝만 바꿔줌

- ### SPA장점은?

  하나의 페이지
  서버회 1회 리소스 요청
  이후에 데이터만 가져와 기존 페이지 수정
  --> 그래서 화면바뀌지 않아 자연스러움

- ### SPA단점?

  SEO에는 약함..
  MPA는  html요소가 다 나눠져 찾기 쉽지만 SPA는 한 페이지.
  이를 보완키 위해 react base로 next.js 씀

- ### SPA 프레임워크 종류 : SPA, VueJS, AngularJS

- ### 왜 React를 써야되나 : 많이써서..
