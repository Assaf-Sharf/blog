# 들어가며
- 회사에서 한 작업을 컨플루언스에 적어야 하는데 회사 컨플루언스는 내부 인트라넷이라 나중에 내가 한 일을 외부에서 확인하기에 어려움이 있었다.
- 이를 해결하기 위해서 `blog`에 그 내용을 적은 뒤 내용만 컨플루언스에 옮기기 위해 github.io를 통해 내 `blog`를 구축했다.
- 하지만 그렇게 해서 만든 페이지의 폰트가 너무 커서 보기가 불편했고, 인터넷과 회사 동기의 정보를 참고하여 폰트 사이즈를 변경할 수 있었다.
- 해당 테마는 **minimal-mistakes** 기준으로 작성되었다.
- 다음에는 `post`는 폰트를 바꾸는 것을 해보고 작성할 예정이다.

# How to?
## /_sass/minimal-mistakes/_reset.scss 수정
자신의 레파지토리의 /_sass/minimal-mistakes/_reset.scss를 확인한다.
``` scss
html {
  /* apply a natural box layout model to all elements */
  box-sizing: border-box;
  background-color: $background-color;
  font-size: 16px;

  @include breakpoint($medium) {
    font-size: 18px;
  }

  @include breakpoint($large) {
    font-size: 20px;
  }

  @include breakpoint($x-large) {
    font-size: 20px;
  }

  -webkit-text-size-adjust: 100%;
  -ms-text-size-adjust: 100%;
}
```
해당 파일을 살펴보게 되면 바꾸지 않았다면 위에 코드처럼 되었있다.
breakpoint 환경에 반응하여 폰트의 사이즈를 유동적으로 변경하기 위해 사용된다.
위에 코드에서도 알 수 있듯이 `font-size:16px`로 되어있다.
즉, 기본 폰트 사이즈(모바일)는 16px이고, 사용자의 창의 크기가 `medium(HD)`, `large(FHD)`, `x-large(QHD)`로 변경될 때마다 각자에 설정된 폰트 사이즈로 변경되는 것이다.
> 모두 동일한 폰트로 보고 싶어!

라는 생각을 가지고 있다면 모두 동일한 폰트 사이즈로 바꾸면 된다.
하지만, 상황에 맞게 폰트를 다르게 하고 싶다면 자신이 원하는 폰트로 바꾸면 된다.

나는 폰트를 바꾸기 위해 참고한 사이트의 폰트 사이즈를 그대로 사용한다.
``` scss
html {
  /* apply a natural box layout model to all elements */
  box-sizing: border-box;
  background-color: $background-color;
  font-size: 14px;

  @include breakpoint($medium) {
    font-size: 16px;
  }

  @include breakpoint($large) {
    font-size: 16px;
  }

  @include breakpoint($x-large) {
    font-size: 16px;
  }

  -webkit-text-size-adjust: 100%;
  -ms-text-size-adjust: 100%;
}
```
이렇게 적용해보니 나쁘지 않았다.
