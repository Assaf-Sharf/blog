## 4계층 프로토콜
### 역할
전송 계층은 송신자와 수신자의 프로세스를 연결하는 통신 서비스를 제공한다.  
전송 계층은 연결 지향 데이터 스트림 지원, 신뢰성, 흐름 제어, 다중화 등의 서비스를 제공한다.

### TCP
TCP는 안전한 연결을 지향한다.

### UDP
UDP는 안전한 열결보다 빠른 연결을 지향한다.

## 포트 번호
### 특징
특정 프로세스들끼리 통신하기 위해서 사용한다.  
하나의 프로세스가 여러 개의 포트를 사용하는 것은 가능하지만 여러 프로세스가 하나의 포트를 사용하는 것은 안된다.  
포트 번호는 일반적으로 정해져 있지만 무조건 지켜야 하는 것은 아니다.
  - 예로 웹 서비스는 80번 포트를 사용하지만 항상 사용하는 것은 아니다.

### Well-known 포트
잘 알려진 포트

|서비스 이름|포트 번호|
|----------|--------|
|FTP|20, 21|
|SSH|22|
|TELNET|23|
|DNS|53|
|DHCP|67, 68|
|TFTP|69|
|HTTP|80|
|HTTPS|443|

### Dynamic 포트
일반 사용자가 사용하는 포트  
49152번 ~ 65535번
