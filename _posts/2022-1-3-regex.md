## 1. [0-9]+ : 연속된 숫자 매칭


```python
regex = '[0-9]+'

search_target = 'abc123xyz'
```

## 11. (…) : Capture Group


```python
regex = '^(file.+)\.pdf$'

search_target = 'file_record_transcript.pdf'

# re.match() 함수로 group화
m = re.match(regex, search_target)
```


```python
# 선택한 그룹
m.group(1)
```

## 12. (a(bc)) : Capture - Sub-group


```python
regex = '(\w+ (\d+))'

search_target = 'Jan 2021, we participated in party'

# re.match() 함수로 group화
m = re.match(regex, search_target)
```


```python
# 선택한 그룹
print(m.group(0))
print(m.group(1))
print(m.group(2))

```
