> 백준 25304  /  영수증

```python
sum = int(input())
n = int(input())
a=0
b=0
t=0

for i in range(n):
    a,b = map(int, input().split())
    t += (a*b)
if t == sum:
    print('Yes')
else:
    print('No')
```
