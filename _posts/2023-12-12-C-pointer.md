# 포인터

- - -

## 포인터란?

메모리의 주소값을 저장하는 변수

- 주소 연산자(&) : 해당 변수의 메모리 시작 주소값을 반환, (&: ampersand) (~의 주소)
- 간접 참조 연산자(*) : 선언된 포인터 변수가 가리키는 변수를 구함 (~의 값)
- 포인터(*) : 주소를 저장하고 있는 변수, 포인터 변수 선언할 때 사용
