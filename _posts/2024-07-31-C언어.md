# 동적할당함수 
프로그램 실핼 중에 저장 공간을 할당하는 것을 동적 할당이라고 합니다. 

## malloc, free함수
프로그램 실행 중에 메모리를 동적할당할 때는 malloc 함수를, 반환할 때는 free함수를 사용합니다. 

```C
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
	int* pi;
	double* pd;

	pi = (int*)malloc(sizeof(int));
	if (pi == NULL)
	{
		printf("# 메모리가 부족합니다.\n");
		exit(1);
	}

	pd = (double*)malloc(sizeof(double));

	*pi = 10;
	*pd = 3.4;

	printf("정수형으로 사용: %d\n", *pi);
	printf("실수형으로 사용: %.1lf\n", *pd);

	free(pi);
	free(pd);

	return 0;
}
```
