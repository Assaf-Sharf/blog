# Access Token & Refresh Token

기본 JWT 방식의 인증(보안) 강화 방식인 Access Token & Refresh Token을 알아보기 전,
서버가 클라이언트 인증을 확인하는 방식과 그 중 하나인 JWT(Json Web Token) 에 대한 이해가 필요하다.

## 클라이언트 인증 방식

보통 서버가 클라이언트 인증을 확인하는 방식은 대표적으로 쿠키, 세션, 토큰 3가지 방식이 있다.
