## 키워드 매개변수 **kwargs
 - 매개변수 앞에 ** 두 게 붙인 케이스
 - **kwargs는 가변 개수의 키워드 인수를 함수에 전달할 때 사용한다.
 - 이것은 **딕셔너리 형태**로 처리된다.


```python
#예제 - AI가 생성한 코드
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="지민", age=25, city="서울")

```

    name: 지민
    age: 25
    city: 서울
    


```python
def print_kwargs(**kwargs) :
    #이 메서드에는 return 값이 없음
    print(kwargs)
```


```python

```

    {'a': 1, 'b': [1, 2, 3]}

## class
### why class?
 - 함수, 반복문만 적절하게 사용해도 큰 문제 안됨
   + 한 줄로 요약 : **자동화** , 많은 사람들이 사용할 수 있도록 **베포**


  - 코드의 **재사용성**: 클래스를 사용하면 코드를 한 번 작성하고 여러 번 재사용할 수 있습니다. 예를 들어, ‘사람’ 클래스를 만들고 이름, 나이, 성별 등의 속성을 정의하면, 이 클래스를 사용하여 수많은 ‘사람’ 객체를 생성할 수 있습니다.
  - 
캡슐 화: 클래스는 데이터와 함수를 하나의 단위로 묶는 데 사용됩니다. 이를 통해 데이터를 숨기고 함수를 통해서만 접근할 수 있도록 제한할 수 있습니다. 이는 데이터의 무결성을 보장하고 코드의 안정성을 높입니다
  - 
 상속: 클래스를 사용하면 기존 클래스의 속성과 메소드를 상속받아 새로운 클래스를 생성할 수 있습니다. 이를 통해 코드의 재사용성을 높이고, 복잡성을 줄일 수 있습니
  - .
 다형성: 클래스를 사용하면 같은 이름의 메소드를 가진 여러 클래스를 동일하게 처리할 수 있습니다. 이를 통해 코드의 유연성을 높일 수 있습니다.

### 만드는 순서

1. class 키워드를 사용하여 클래스를 정의.
2. 클래스 이름을 작성. 클래스 이름은 보통 대문자로 시작하는 카멜케이스(CamelCase)를 사용한다.
3. 콜론(:)을 사용하여 클래스 정의를 시작한다.
4. 클래스 내부에 **메소드**를 정의한다. 메소드는 함수와 비슷하지만, 클래스 내부에 정의되며 **첫 번째 인수로 [self*](https://wikidocs.net/1742) 를 받는다.

### 예제 소스

```python
    def greet(self):                #greet라는 메소드정의 / self를 받는다.  
        print("안녕하세요!")

# 클래스의 인스턴스 생성              MyClass의 인스턴스 (my_instance)생성
```

### class 상태와 행동
 - key point : 내가 사용하는 함수가 상태인가 행동인가를 인지하는 것.


```python
class Student:
    # 상태
    def __init__(self, name, korean, math, english, science):
        self.name = name             
        self.korean = korean          # 87
        self.math = math              # 98
        self.english = english        # 88 
        self.science = science        # 95
    
    # 행동
    def get_sum(self):
        return self.korean + self.math + self.english + self.science

    def get_average(self):
        return self.get_sum() / 4
        
    def get_grade(self): 
        grade = None
        if self.get_average() > 90:
            grade = 'A'
        else:
            grade = 'F'

        return grade

    def __str__(self): 
        return "{}\t{}\t{}\t".format(self.name, self.get_sum(), self.get_average())
```


```python
students = [
    Student("윤인성", 87, 98, 88, 95),
```
