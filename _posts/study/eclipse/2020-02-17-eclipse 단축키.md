# 단축키

## Search : 찾기

- 특정 메소드를 호출한 모든 위치를 확인할 수 있다.

	`Ctrl` + `Alt` + `H`  
	
- 원하는 resource들을 찾을 수 있다. ( java, xml 등 )

	`Ctrl` + `Shift` + `R`


## Folding : 소스코드 열/접기

기본적으로 Folding 기능은 메소드, 주석(한줄 주석 제외), import문이 주어진다.

  

마우스로 하는 방식은 위에서 초입부에서 언급했으니, 바로 단축키 사용법을 설명하도록 하겠다.

  

-   단일 Folding 접기

    단축키 Ctrl + Numpad_Plus(+)




-   단일 Folding 펼치기
    
    단축키 Ctrl + Numpad_Plus(+)
    

  

-   전체 Folding 접기
    
    단축키 Ctrl + Shift + Numpad_Divide(/)
    
    ![](https://t1.daumcdn.net/cfile/tistory/991EA63B5C12773B2C)
    

  

-   전체 메소드 펼치기 및 메소드 외 Folding 접기
    
    단축키 Ctrl + Shift + Numpad_Asterisk(*)
    
    ![](https://t1.daumcdn.net/cfile/tistory/994BD9375C12777812)
