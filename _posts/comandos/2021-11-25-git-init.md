## Git init

* Ejecutando este comando ``git init`` creas un directorio oculto llamado ``.git`` que transforma un **directorio normal** en un directorio de proyecto **[Repo.Local]** para poder trabajar con el sistema de versiones de **Git** y **Github** sin problemas

  * El subdirectorio oculto **.git** es una especie de esqueleto que contiene entre otras cosas la base de datos donde guardar todos los cambios del proyecto , **commits** , **ramas** y demás elementos que permiten el correcto funcionamiento de **GIT**

  * Por defecto cuando ejecutas el comando ``git init`` crea la ``rama master`` pero desde la versión 2.28 se puede cambiar el nombre de la rama por defecto mediante el comando:

``git config --global init.defaultBranch <nombre-de-la-rama>``

Ejemplo

``git config --global init.defaultBranch mi-primera-rama``
