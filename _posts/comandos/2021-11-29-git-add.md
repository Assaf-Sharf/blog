## Git - add

Almacena el archivo o todos los archivos dentro del estado **{INDEX / Staging Area}** a la espera de ser enviados al [Repositorio Remoto] después de hacer un ``git commit -m "mensaje"`` y confirmar sus cambios o creación para el proyecto

* Añadir un archivo al {INDEX/Staging Area} a la espera de ser confirmados mediante un ``git commit -m "mensaje"``  para su envio al «Repositorio Remoto»

```git
git add <archivo>
```

* Añadir todos los archivos al {INDEX/Staging Area}

```git
git add .
```
