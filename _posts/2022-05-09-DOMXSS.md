# DOM XSS란?
DOM = Document Object Model  
XSS = Cross-site Scripting  
클라이언트 측에서 사용하는 브라우저의 DOM 환경을 수정하여 공격하는 방식   
DOM 환경을 수정하려면 html 안에 특정 소스를 추가해서 할것이다  

<br>
