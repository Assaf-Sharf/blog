#  1.  라즈비안 OS 설치된 SD 카드 열기

# 2. 라즈비안 파일 wpa_supplicant.conf 파일 넣기

wpa_supplicant.conf 안에 내용

```
country=GB
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
network={
        ssid="와아피아 이름"
        psk="와아파이 비번"
}
```


# 3.  라즈베리파이 전원 넣고 IP 찾으면 된다.
