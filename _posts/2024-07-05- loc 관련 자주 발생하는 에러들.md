에러를 일일이 검색하는 것에 큰 시간낭비를 방지하고자 함이 내 목표이다.
이것을 정리하며 오류에 공부하니 정답을 바로 공부하는 것만큼 도움됐다.

나중에 내가 참고하려고 정리한 것.

### Series.isin() takes 2 positional arguments but 3 were given

2. isin() 안 괄호 안에 대괄호로 한 번 더 묶기 


```python
ns_book7.loc[ns_book7['대출건수'].isin(4,5)]
```

### unsupported operand type(s) for &: 'float' and 'bool'

1. 중첩된 df명 지우기

error 1. 


```python
c1 = df[(df['age'] >= 30)] 
c2 = df[(df['sex'] == 'male')]
df.loc[(c1) & (c2)].sort_values(['fare'], ascending= False)
```

error 2.


```python
c1 = df[df['age'] >= 30] 
c2 = df[df['sex'] == 'male']
df.loc[(c1) & (c2)].sort_values(['fare'], ascending= False)
```

⇊


```python
c1 = df['age'] >= 30
c2 = (df['sex'] == 'male')   #각각의 조건 (c1,c2)에서 전체를 ()로 묶는 것은 상관없음
df.loc[(c1) & (c2)].sort_values(['fare'], ascending= False)
```

## AttributeError

### 'list' object has no attribute 'str'

3. df명을 컬럼명 앞 대괄호에 한 번 더 쓰기


```python
ns_book7.loc[['저자'].str.contains('류시화')].reset_index(drop=True)
```

⇊


```python
ns_book7[ns_book7['저자'].str.contains('류시화')].reset_index(drop=True)
```

4. **":"** 위치 바꿔보기


```python
ns_book7.loc[ : , ['저자'].str.contains('류시화')]
```

⇊


```python
ns_book7.loc[ns_book7['저자'].str.contains('류시화'), : ]
```

## 대괄호 [] 와 소괄호()

### [ ] (대괄호):

- 열 선택: df['column_name'] 형태로 사용하면 'column_name’이라는 이름의 열을 선택할 수 있다.

- 행 선택: 불리언 인덱싱(boolean indexing)을 사용하여 특정 조건을 만족하는 행을 선택할 수 있다. 예를 들어, df[df['column_name'] > 0]은 ‘column_name’ 열의 값이 0보다 큰 모든 행을 선택한다.

- 슬라이싱: df[1:5]와 같이 사용하면 행을 슬라이싱할 수 있다. 이 예제는 인덱스 1부터 4까지의 행을 선택한다.


### ( ) (소괄호):

- 함수 호출: ()는 함수를 호출할 때 사용된다. 
예를 들어, df.head(5)는 데이터프레임의 처음 5행을 반환한다.

- 연산 순서 지정: ()는 연산의 순서를 지정하는 데 사용된다.
- 예를 들어, df[(df['column1'] == 'value1') & (df['column2'] > 0)]은 'column1’이 'value1’이고 'column2’가 0보다 큰 모든 행을 선택한다.
- 여기서 ()는 논리 연산자 &가 각 조건을 개별적으로 평가하도록 한다.
