## **백준 10430 “나머지”**
<br>

### **문제**
---
(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?

(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?

세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

<br>

### **입력**
---
첫째 줄에 A, B, C가 순서대로 주어진다. (2 ≤ A, B, C ≤ 10000)

<br>

### **출력**
---
첫째 줄에 (A+B)%C, 둘째 줄에 ((A%C) + (B%C))%C, 셋째 줄에 (A×B)%C, 넷째 줄에 ((A%C) × (B%C))%C를 출력한다.

<br>

### **입력예제**
---
```python
# 입력 : 5 8 4
# 출력 : 1 
#       1
#       0
#       0
```

<br>

### **문제해결**
---
```python
A, B, C = map(int, input().split())
print((A+B)%C)
print(((A%C)+(B%C))%C)
print((A*B)%C)
print(((A%C)*(B%C))%C)
```  
  
<br>

### **공부한 문법**
---
1. 수학식 계산 "%"  
"/" (기본형 나눗셈 - float)  
"//" (나눗셈의 몫)  
"%" (나눗셈의 나머지)
  
ex :
```python
a = 5
b = 3
a / b = 1.66666666667
a // b = 1
a % b = 2
```
