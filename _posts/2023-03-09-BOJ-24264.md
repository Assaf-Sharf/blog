## 문제  
24264

## solution  
```
MenOfPassion(A[], n) {
    sum <- 0;
    for i <- 1 to n
        for j <- 1 to n
            sum <- sum + A[i] × A[j]; # 코드1
    return sum;
}
```
첫째 줄에 코드1 의 수행 횟수를 출력한다.  
둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## CODE

```python
print(int(input())**2,2,sep='\n')
```
문제의 알고리즘은 n에 따라서, 중첩된 for문이 n^2번 실행되므로, 빅오 표기법으로 O(n^2)이다. 수행횟수는 n^2번, 최고차항의 차수는 2(n^2)
