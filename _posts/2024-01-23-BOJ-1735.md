## 문제  
1735

## solution  
분수 덧셈

## CODE  

```python
import math
a1,a2 = list(map(int,input().split()))
b1,b2 = list(map(int,input().split()))
a1 *= b2
b1 *= a2
a2 *= b2
a1 = a1 + b1
print(int(a1/math.gcd(a1,a2)),int(a2/math.gcd(a1,a2)))
```
math.gcd을 꼭 써야 하나?  
나눗셈의 몫으로 해결 가능해보인다.  
    
```python
[a,b],[c,d]=eval('map(int,input().split()),'*2);u=p=a*d+b*c;v=q=b*d
while p:q,p=p,q%p
print(u//q,v//q)
```
받아오는 부분을 줄여서 쓸수 있다.
