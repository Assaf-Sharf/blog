### **9. Index of****를** **if****문에** **쓸때** **주의사항****!**

```js	
function hasCola(str) {
  if (str.indexOf("콜라") > -1) {
    console.log("금칙어가 있습니다.");
  } else {
    console.log("통과");
  }
}
hasCola("와 사이다가 짱이야!");
hasCola("무슨소리, 콜라가 최고");
hasCola("그래 콜라!");
```
