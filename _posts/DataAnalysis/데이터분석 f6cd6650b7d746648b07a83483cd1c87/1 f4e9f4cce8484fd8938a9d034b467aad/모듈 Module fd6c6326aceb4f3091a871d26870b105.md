# 모듈 Module

- [https://docs.python.org/ko/3/installing/index.html](https://docs.python.org/ko/3/installing/index.html)

## 1. 모듈 (Module) 이란?

```python
다른 파이썬 프로그램에서 불러와 사용할 수 있게 함수나 변수 또는 클래스를 모아 놓은 파일
```

- 다른 사람들이 이미 만들어 놓은 모듈을 사용할 수도 있고 직접 만들어서 사용할 수도 있다.

## 2. 모듈 설치하기

```python
python -m pip install SomePackage
```

- 정확한 버전이나 최소 버전을 명령줄에 직접 지정할 수도 있다
- `>`, `<` 와 같은 비교 연산자나 셸에서 해석되는 다른 특수 문자를 사용할 때는, 패키지 이름과 버전을 큰따옴표로 묶어야 한다.
    
    ```python
    python -m pip install SomePackage==1.0.4    # specific version
    python -m pip install "SomePackage>=1.0.4"  # minimum version
    
    python -m pip install --upgrade SomePackage
    ```
    
    Function < Module < Package < Library
