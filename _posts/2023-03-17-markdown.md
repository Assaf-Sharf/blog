# 블로그 완성 기념 markdown 문법 정리!



## heading

/# Heading(1)

/ ##  Heading (2)

/ ### Heading (3)

/ ####  Heading (4)

/ ##### Heading (5)

/ ######  Heading (6)



## bold

/ ** 단어 **

ex)   **단어**





## italic

/  * 단어 *

ex) *단어*



## strikethrough

/ ~~ 단어 ~~

ex) ~~단어~~



## qoute

/ > 단어

ex) > 단어



# bullet list
