# 다른곳으로의 링크



| **target** **속성값** | **설명**       | **비고**                                     |
| --------------------- | -------------- | -------------------------------------------- |
| _self                 | 현재 창        | 기본                                         |
| _blank                | 새 창          | 텍스트나 내부 이미지의 alt  등으로 명시 필요 |
| _parent               | 부모  프레임   | <iframe> 사용시                              |
| _top                  | 최상위  프레임 | <iframe> 사용시                              |



```html
<a href="연결할 주소" target="링크를 열 옵션">
```
