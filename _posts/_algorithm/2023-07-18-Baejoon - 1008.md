## 내 풀이

**첫번째 오답**

```java
public java.util.Scanner
    
    public class Main{
        public static void main(String []args){
            Scanner sc = new Scanner(System.in);
            
            double = A,B;
            
             A = (int)(sc.nextInt());
             B = (int)(sc.nextInt());
            
            System.out.println(A/B);
   
        }
    }
```

결과는 에러가 뜨는데, 여기서 A,B의 변수를 double로 데이터 타입을 잡았는데 Scanner의 메소드를 nextDouble이 아닌 nextInt로 메소드를 사용하였기 때문에 에러가 떴다. <br/>



**두번째 오답**

```java
public java.util.Scanner
    
    public class Main{
        public static void main(String []args){
            Scanner sc = new Scanner(System.in);
            
            int A,B;
            int A = sc.nextInt();
            int B = sc.nextInt();
            
            System.out.println((double)(A/B));
   
        }
    }
```

(double)(a / b)로 작성하면 (a / b)가 이미 정수의 형태로 계산된 뒤 실수 형태로 변환되기 때문에 오차가 발생한다. 그래서 먼저 실수로 변환 될 수 있도록 (double) a / b로 작성하자!

<br/>





## 정답

**내가 생각한 정답**

```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args){
 
  Scanner sc = new Scanner(System.in);
  int A,B;
     
  A = sc.nextInt();
  B = sc.nextInt();
  System.out.println((double)A/B);

 }
}
```

먼저 int 데이터 타입을 사용해서, 정수를 입력 받은 다음, 캐스팅(형변환)을 하여 실수형태로 값이 표현되도록 답을 작성하였다.

<br/>



**다른사람 정답**

```java
import java.util.Scanner;

public class Main {
  public static void main(String[] args){
 
  Scanner sc = new Scanner(System.in);
     
  double A = (double)sc.nextInt();
  double B = (double)sc.nextInt();
  System.out.println(A/B);

 }
}
```

nextInt메소드를 사용하여 입력되는 데이터 값을 정수로 받고 double로 캐스팅하여 실수로 표현되도록 하는 방법이다.

<br/>



**다른 사람 정답**

```java
import java.util.Scanner;

public class Main {
 public static void main(String[] args){
 
  Scanner sc = new Scanner(System.in);
     double = A,B;
     
     A = sc.nextDouble();
     B = sc.nextDouble();
     System.out.println(A/B);

 }
}
```

Scanner메서드 중에서 double값을 읽을 수 있는 메서드가 있는 줄 몰랐다.. 인터넷에 찾아보니, 자바 기본 데이터 타입의 모든 종류를 Scanner 통해 값을 받아올수 있는 것을 확인했다.

<br/>





**Scanner 메소드**

|     메서드     | 기능                                                         |
| :------------: | ------------------------------------------------------------ |
|   **next()**   | **String**을 읽음. 단, **띄어쓰기 뒷부분은 읽지 않음**.      |
| **nextLine()** | **String**을 읽음. **띄어쓰기를 포함하여 한 줄(즉, Enter를 치기 전까지)을 읽음**. |
| **nextInt()**  | **int** 를 읽음.                                             |
| nextBoolean()  | boolean 을 읽음                                              |
|   nextByte()   | byte 를 읽음                                                 |
|  nextShort()   | short 를 읽음                                                |
|   nextLong()   | long 을 읽음                                                 |
|  nextFloat()   | float 을 읽음                                                |
|  nextDouble()  | double 을 읽음                                               |
