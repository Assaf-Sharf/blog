> ### 기존에 접속했던 와이파이 비밀번호 확인하기 윈도우

- 윈도우 cmd 창 열기
-> 윈도우 실행창에 cmd 실행
- 아래 명령어 입력
```
netsh wlan show profiles
```
- 위 명령어 입력시 여태까지 접속했던 와이파이 리스트 이름이 나옴
```
netsh wlan show profiles [wifi 이름] key=clear
```
- 위 명령어 입력 시 보안설정 부분 맨 아래에 키 콘텐츠가 비밀번호
- **단, 접속했던 와이파이여야함**
