#코드 결과

```python
n, k =map(int, input().split())<br>
cnt =0<br>
for i in range(1, n+1):<br>
    if n%i==0:<br>
        cnt+=1<br>
    if cnt==k:<br>
        print(i)<br>
        break<br>
else:<br>
    print(-1)<br>
```
