# 알고리즘 오답노트
분수의 덧셈  
## 문제:첫 번째 분수의 분자와 분모를 뜻하는 numer1, denom1, 두 번째 분수의 분자와 분모를 뜻하는 numer2, denom2가 매개변수로 주어집니다. 두 분수를 더한 값을 기약 분수로 나타냈을 때 분자와 분모를 순서대로 담은 배열을 return 하도록 solution 함수를 완성해보세요.
## 예시
입출력 예 설명  
입출력 예 #1  
1 / 2 + 3 / 4 = 5 / 4입니다. 따라서 [5, 4]를 return 합니다.  
입출력 예 #2  
9 / 2 + 1 / 3 = 29 / 6입니다. 따라서 [29, 6]을 return 합니다.  
### 어려웠던 부분
분자와 분모가 주어졌기 때문에 덧셈을 구하는 것은 쉬웠다.  
분자는 numer1*denom2+denom1*numer2  
분모는 denom1*denom2  
문제는 기약분수로 만드는 부분이였다.
예를 들자면 15/30이라는 결과가 나왔다면 최대공약수인 15로 나워서 1/2라는 해답을 내놔야 한다.  
최대공약수...  
최대공약수에서 막혔다.  
결국엔 인터넷을 참고를 해서 다음과 같은 코드를 짰다   
```javascript
function solution(numer1, denom1, numer2, denom2) {
    let answer = [];
    let num = numer1*denom2+denom1*numer2;
    let denom = denom1*denom2;
    let snum = num<denom?num:denom;
    let G;
    for (let i = 1; i <= snum; i++) {
        if (num % i == 0 && denom % i == 0) {
            G = i;
        }
    }
    answer = [ num/G, denom/G ]
    return answer;
}
```
분모와 분자 중에 작은 수를 snum이라고 정하고  
for루프를 통해서 분자와 분모로 나누어 떨어지는 snum보다 작은 수를 G로 한다  
이 루프가 끝났을 때의 G가 최대공약수가 될것이다.  
이를 통해서 결과를 도출해낼 수 있었다
