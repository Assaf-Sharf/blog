## 문제  
2444

## solution  
N이 주어지면 첫째 줄부터 2*N-1번째 줄까지 차례대로 별을 출력한다.  
가운데 줄의 별 개수도 2*N-1개

## CODE  

```python
tmp = int(input())
for k in range(1,tmp+1):
    print(' '*(tmp-k) + '*'*(k*2 - 1))
for k in range(1,tmp):
    print(' '*k + '*'*((tmp-k)*2 - 1))
```
제출한 코드. 조금 더럽게 풀었다는 생각이 든다.

```python
n=int(input())
for i in map(abs,range(1-n,n)):print(' '*i+'*'*((n-i)*2-1))
```
절댓값을 이용하여 깔끔하게 범위를 잡을 수 있다.
