# Error
- LightGBM 을 설치하기 위해 terminal 에 아래 코드를 입력했다.
- 코드를 입력했더니 엄청난 양의 오류 메시지가 출력되었다.
- [🔍 오류 메시지 보러가기](https://github.com/j-jae0/Issue/blob/main/lightgbm_install_issue.md)

## PIP 설치 - 오류 발생
```
pip install lightgbm
```

## CONDA 설치 - 오류 해결
```
conda install -c conda-forge lightgbm
```

## 회고
>끝없이 발생하는 오류 메시지를 보며,, 설마 M1 이슈인가..! 해서 관련 정보를 찾아보았는데 brew, xcode … 등 으로 설치해야된다는 글을 발견했다.<br>
Home brew 때문에 머리아팠던 적이 있어서 다른 방안을 모색하다가 아나콘다 환경에 설치하는 방법을 사용해보았다.😱<br>
결과는 아나콘다 환경에서 설치 성공하였다!<br>
<u>pip으로 설치했을 때 에러가 발생하면 바로 conda에 설치하자</u>
