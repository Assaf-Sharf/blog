>> 따라서 오늘은 백준문제를 풀어보기로 했다. 사실 백준문제에는 JavaScrip를 지원하지 않기 때문에 프로그래머스를 활용해야 했다.

### <center>오늘의 백준 문제 
시험점수를 정수로 바꾸는 문제 

```
시험 점수를 입력받아 90 ~ 100점은 A, 80 ~ 89점은 B, 70 ~ 79점은 C, 60 ~ 69점은 D, 
나머지 점수는 F를 출력하는 프로그램을 작성하시오.
```
### <center> 나의 문제 풀이 
```javascript
> 프로그램 : Javascript
>> Code : 
let score = prompt('enter your score')
if (score >= 90){
    console.log('A')
} else if(score >= 80){
    console.log('B')
} else if(score >= 70){
    console.log('C')
} else if(score >= 60){
    console.log('D')
}else{
    console.log('F')
} 

```
### <center>추가적으로 공부해야 할 점 
```
이 문제가 어려운 문제가 아닌 쉬운 문제임에도 불구하고 숫자 범위설정하는 부분에 있어서 상당한 시간이 걸렸다. 

시간을 좀 더 줄일 필요가 있고 기본기를 좀 더 확실히 다지는 공부를 시작하자 

물론 기본기 다지기 공부를 하면서 백준 기본 문제를 계속해서 풀어보자
```
