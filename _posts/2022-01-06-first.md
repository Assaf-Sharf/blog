# 오늘 처음 블로그를 만들었어요

- 앞으로 열심히 해보겠습니다.

- 연습으로 파이썬 코드 붙여넣기
```python
>>> my_list = [1, 2, 3, 4, 5]

>>> for elem in my_list:
	if elem % 2 == 0:
		print("Even:", elem)
		print("break")
		break
	else:
		print("Odd:", elem)

		
Odd: 1
Even: 2
break
```
