### 리스트 복사

그냥 b = a 하면

같은 주소를 지정하게 되어 a의 값들을 바꾸면 b도 같이 바뀌게 된다.

- 1차원

```python
b= a[:]
```

를 하면 같은 주소를 지정하지 않고도 리스트를 복사할 수 있음 (1차원에서만 가능)

- 2차원

```python
import copy
b = copy.deepcopy(a)
```

### 콘솔창 입출력

1. input()

   input ("~~ ")  : 사용시 ~~ 출력후 입력 받음

2. print()

​		출력형식

​		1) %string  %('~')

​				*padding

​					ex) %10.3f : 10칸을 비우고 3칸은 소수로 사용

​		2) ''{}''  .format

​		3) f-string				

```python
name = "Suntae"
age = 39

print(f"My name is {name}\, and I'm {age}") 

print(f'{name:20}')
#20칸 비워놔
print(f'{name:>20}')
#20칸 비워놓고 오른쪽 정렬
print(f'{name:*<20}')
#20칸 비워놓고 *로 채워 왼쪽 정렬
print(f'{name:*^20}')
#20칸 비워놓고 *로 채워 가운데 정렬

```
