title: Google Chrome version 70 is out, bringing with it support for AV1 and Progressive Web Apps
date: October 18, 2018 at 02:17AM
categories: Tech News
---
<img class="align-center" src="%20http://d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&amp;height=36&amp;mode=crop&amp;upscale=false">
<p><p>Google has released version 70 of its flagship Chrome web browser to its stable update channel, bringing with it support for several technologies.</p>
<p>This update introduces a new media codec for all desktop operating systems that support <a href='//alternativeto.net/software/google-chrome/'><img alt='Small Google Chrome icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&height=36&mode=crop&upscale=false' />Google Chrome</a> called AV1. It also introduces support for Progressive Web Apps for the Windows version. Support for Progressive Web Apps in the <a href='//alternativeto.net/software/mac-os/'><img alt='Small macOS icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/mac-os_132940.png?width=36&height=36&mode=crop&upscale=false' />macOS</a> and <a href='//alternativeto.net/software/linux-kernel/'><img alt='Small Linux kernel icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/linux-kernel_97656.png?width=36&height=36&mode=crop&upscale=false' />Linux kernel</a> operating systems releases will be added in later versions.</p>
<p><a href="https://research.mozilla.org/av1-media-codecs/" rel="nofollow">AV1</a> is a new audiovisual codec based off of Google's <a href="https://developers.google.com/media/vp9/" rel="nofollow">VP9 video compression technology</a> and promises significantly smaller file sizes at no loss of quality compared to media encoded via H.264 or H.265. AV1 support for Chrome 70 can be easily tested on YouTube via enabling it via <a href="https://www.youtube.com/testtube" rel="nofollow">its TestTube site</a> and <a href="https://www.youtube.com/playlist?list=PLyqf6gJt7KuHBmeVzZteZUlNUQAVLwrZS" rel="nofollow">this playlist of supported videos</a>.</p>
<p>As <a href="https://www.youtube.com/watch?v=msA284Q6yZU" rel="nofollow">Google posted on its YouTube channel</a>, Chrome 70 also implements support for additional login credentials, such as devices compatible with biometric security options. These include Apple's devices with Touch ID or <a href='//alternativeto.net/software/android/'><img alt='Small Android icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/android_84815.png?width=36&height=36&mode=crop&upscale=false' />Android</a> devices with fingerprint sensors.</p>
