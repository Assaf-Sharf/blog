# 기본적인 문법
1. SHOW DATABASES; - 현재 데이터베이스 보여주
2. USE stevedb;  - 데이터 베이스 사용
3. SHOW TABLES;  -현재 테이블 보여주기
4. SELECT * FROM address; 기본적인 보여주
