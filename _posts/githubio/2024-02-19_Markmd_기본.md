## 주요기능

1. 헤더 #
- 헤더 입력시 # 을 추가해서 글을 쓰면 된다. 
 갯수가 추가되면 하위 헤더 기능으로 추가된다

 2. 인용 >
 > 인용 문구 일시 \> 일력해서 쓰면된다


 3. 목록 1. or *,+,-
 - 순서있는 목록일시 1. 으로 이어쓰면된다
 
 - 순서가 상관이 없다면
   * *입력 될 경우 
   + +일력 될 경우
   - -입력 될 경우 

4.  - 코드 \``` ```

```
 코드 블럭으로 ``` ``` 을 입력 후 안에 코드를 입력하면된다 .

입력시 해당 언어에 맞게 함수 변수 표현식을 보기 편하게 꾸며 준다.
```
