<img class="align-center" src="%20http://d2.alternativeto.net/dist/icons/ublock-origin_66152.png?width=36&amp;height=36&amp;mode=crop&amp;upscale=false">
<p><p>eyeo GmbH has released the latest version of Adblock Plus, 3.4. This release includes a refresh to the user interface and an average 50% reduction in system memory usage.</p>
<p>The new UI removes the social media icons that could be clicked to visit Adblock Plus's social media pages, as well as the total number of ads blocked in total. It adds a list of blocked items on any given webpage versus the page's total number of items, and also gives a simple toggle for disabling ad blocking on a per-site basis.</p>
<p><a href="https://www.ghacks.net/2018/10/31/adblock-plus-3-4-promises-50-memory-use-reduction/" rel="nofollow">According to a test by Martin Brinkmann of gHacks</a>, however, Adblock Plus still uses more system memory than <a href='//alternativeto.net/software/ublock-origin/'><img alt='Small uBlock Origin icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/ublock-origin_66152.png?width=36&height=36&mode=crop&upscale=false' />uBlock Origin</a> even after being updated to version 3.4.</p>
<p>Adblock Plus 3.4 is <a href="https://adblockplus.org/en/download" rel="nofollow">available</a> on <a href='//alternativeto.net/software/google-chrome/'><img alt='Small Google Chrome icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/google-chrome_96565.png?width=36&height=36&mode=crop&upscale=false' />Google Chrome</a>, <a href='//alternativeto.net/software/firefox/'><img alt='Small Mozilla Firefox icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/firefox_114558.png?width=36&height=36&mode=crop&upscale=false' />Mozilla Firefox</a>, and <a href='//alternativeto.net/software/opera/'><img alt='Small Opera icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/opera_81297.png?width=36&height=36&mode=crop&upscale=false' />Opera</a>, but not yet on <a href='//alternativeto.net/software/internet-explorer/'><img alt='Small Internet Explorer icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/internet-explorer_100965.png?width=36&height=36&mode=crop&upscale=false' />Internet Explorer</a>, <a href='//alternativeto.net/software/microsoft-edge/'><img alt='Small Microsoft Edge icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/microsoft-edge_85645.png?width=36&height=36&mode=crop&upscale=false' />Microsoft Edge</a>, or <a href='//alternativeto.net/software/safari/'><img alt='Small Safari icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/safari_133203.png?width=36&height=36&mode=crop&upscale=false' />Safari</a>.</p>
