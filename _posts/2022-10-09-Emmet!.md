# 코드 빠르게 작성하는 emmet



### .(class) , #(id)

```html
div.class{class}
div#id{id}
.div{class}
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div class="class">class</div>
<div id="id">id</div>
<div class="class">class</div>
```

___



### Html 기본 소스: !

```html
! (tab)
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>

```

___



### 자식 : >

```html
div>ul>li
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div>
  <ul>
    <li></li>
  </ul>
</div>
```

___

### 형제: +

```html
div>ul+ol
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div>
  <ul></ul>
  <ol></ol>
</div>
```

___

### 똑같은것 여러개 만들고 싶을때 : *n(숫자)

```html
div>ul*4
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div>
  <ul></ul>
  <ul></ul>
  <ul></ul>
  <ul></ul>
</div>
```

___

### 맨마지막 자식에서 바로 위 부모의 형제를 만들고 싶을때: ^

```html
div>ul>li^ol
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div>
  <ul>
    <li></li>
  </ul>
  <ol></ol>
</div>
```



### 태그 그룹화 하기 : ( )

```html
div(header>ul>li*2>a)+footer>p
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<div></div>
<header>
  <ul>
    <li><a href=""></a></li>
    <li><a href=""></a></li>
  </ul>
</header>
<footer>
  <p></p>
</footer>
```



### 텍스트 입력하기 : { }

```html
p{hello}
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<p>hello</p>
```



### 오름차순 숫자 할당법 : $

```html
p.class${item}*5
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<p class="class1">item</p>
<p class="class2">item</p>
<p class="class3">item</p>
<p class="class4">item</p>
<p class="class5">item</p>
```



### 더미용 텍스트 : lorem+n(숫자)

```html
p>lorem4
```

​                                                                                      ↓↓↓↓↓↓↓

```html
<p>Lorem ipsum dolor sit.</p>
```
