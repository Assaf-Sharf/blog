## 문제  
24267

## solution  
```
MenOfPassion(A[], n) {
    sum <- 0;
    for i <- 1 to n - 2
        for j <- i + 1 to n - 1
            for k <- j + 1 to n
                sum <- sum + A[i] × A[j] × A[k]; # 코드1
    return sum;
}
```
첫째 줄에 코드1 의 수행 횟수를 출력한다.  
둘째 줄에 코드1의 수행 횟수를 다항식으로 나타내었을 때, 최고차항의 차수를 출력한다. 단, 다항식으로 나타낼 수 없거나 최고차항의 차수가 3보다 크면 4를 출력한다.

## CODE  

```python
n = int(input());print(n*(n-1)*(n-2)//6,3,sep='\n')
```
알고리즘의 수행 시간이 n^3에 비례한다.  
1부터 n까지의 숫자중 3가지를 중복없이 뽑아서 크기 순으로 작성하는 경우의 수가 수행횟수랑 같다.
