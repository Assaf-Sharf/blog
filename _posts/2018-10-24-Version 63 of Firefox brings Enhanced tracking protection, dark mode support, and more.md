<img class="align-center" src="%20http://d2.alternativeto.net/dist/icons/firefox_114558.png?width=36&amp;height=36&amp;mode=crop&amp;upscale=false">
<p><p>Mozilla today released version 63 of its flagship Firefox web browser to its stable release channel. This release brings several enhancements and features, most notably Enhanced Tracking Protection and support for Windows 10's dark mode and Siri Shortcuts on iOS 12.</p>
<p>Enhanced Tracking Protection in <a href='//alternativeto.net/software/firefox/'><img alt='Small Mozilla Firefox icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/firefox_114558.png?width=36&height=36&mode=crop&upscale=false' />Mozilla Firefox</a> version 63 is off by default, but can easily be enabled in Options/Preferences -&gt; Privacy &amp; Security. As some site layouts and embedded content depend on elements that Enhanced Tracking Protection can potentially block, there will be an option to disable it on an individual site-by-site basis as well. The primary goal of Enhanced Tracking Protection is to, <a href="https://blog.mozilla.org/blog/2018/10/23/latest-firefox-rolls-out-enhanced-tracking-protection/" rel="nofollow">according to Nick Nguyen at Mozilla</a>, &quot;effectively block the most common form of cross-site tracking.&quot;</p>
<p>Other notable additions to this latest stable release of Firefox include support for <a href='//alternativeto.net/software/windows-10/'><img alt='Small Windows 10 icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/windows-10_82838.png?width=36&height=36&mode=crop&upscale=false' />Windows 10</a> users that have selected dark mode as the default app mode in their Windows Settings. Firefox will now match users that have selected this mode with a darker home page and user interface. Additionally, <a href='//alternativeto.net/software/ios/'><img alt='Small iOS icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/ios_88776.png?width=36&height=36&mode=crop&upscale=false' />iOS</a> 12 users will be able to open a new tab in Firefox using <a href='//alternativeto.net/software/siri-shortcuts/'><img alt='Small Siri Shortcuts icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/siri-shortcuts_133367.jpg?width=36&height=36&mode=crop&upscale=false' />Siri Shortcuts</a> voice commands, with more voice commands coming in the future.</p>
