## 문제  
11718

## solution  
받은 문자열 그대로 출력하기. 내용이 여러줄이다.

## CODE  

```python
import sys
input = sys.stdin.read
print(input())
```
제출한 코드. print 안에 sys.stdin.read 그대로 쓰면 되는데 버릇이 들어서 세줄로 타이핑 했다.  

```python
print(open(0).read())
```
open(0)에서  
0은 stdin 1은 stdout 2는 stderr에 해당한다.
