# DWIN 모니터 시리얼 업데이트 방법

1. 원하는 변경 파일을 생성한다 (42.icl(이미지) )

2. 업데이트
32Kbyte 블록으로 나눠서 업데이트를 진행한다.


1.UltraEdit 프로그램으로 이미지 파일 열어준
2.모든 데이터를 선택 후 복사한다 (Ctrl +Shift+ c)
3. 메모장으로 복사한다 

처음 주소 위치는 
42 * 8 형식으로 구한
150 
240 byte 단위로 쪼개준다
