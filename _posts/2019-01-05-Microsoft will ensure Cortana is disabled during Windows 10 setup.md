title: Microsoft will ensure Cortana is disabled during Windows 10 setup
date: January 05, 2019 at 02:22AM
categories: Tech News
---
<img class="align-center" src="%20http://d2.alternativeto.net/dist/icons/cortana_118198.png?width=36&amp;height=36&amp;mode=crop&amp;upscale=false">
<p><p>School administrators and IT admins for enterprises and educational facilities will soon be able to breathe a massive sigh of relief: Microsoft is muting Cortana during Windows 10 setup.</p>
<p><a href='//alternativeto.net/software/cortana/'><img alt='Small Cortana icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/cortana_118198.png?width=36&height=36&mode=crop&upscale=false' />Cortana</a>, the virtual assistant built into Windows 10 similar to the likes of <a href='//alternativeto.net/software/siri/'><img alt='Small Siri icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/siri_118509.png?width=36&height=36&mode=crop&upscale=false' />Siri</a>, <a href='//alternativeto.net/software/google-assistant/'><img alt='Small Google Assistant icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/google-assistant_111549.png?width=36&height=36&mode=crop&upscale=false' />Google Assistant</a>, and <a href='//alternativeto.net/software/amazon-echo/'><img alt='Small Amazon Echo (Alexa Voice Service) icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/amazon-echo_86399.webp?format=png&width=36&height=36&mode=crop&upscale=false' />Amazon Echo (Alexa Voice Service)</a>, had a nasty habit of speaking up during initial setup of all versions of Windows 10. What this meant for IT administrators setting up multiple machines can be described as <a href="https://www.youtube.com/watch?v=Rp2rhM8YUZY" rel="nofollow">nothing short of horrifying</a>. This will soon no longer be the case.</p>
<p>According to Microsoft's official <a href="https://blogs.windows.com/windowsexperience/2019/01/03/announcing-windows-10-insider-preview-build-18309/" rel="nofollow">Windows Experience Blog post about the latest Windows 10 Insider Preview Build</a>, the following change will be implemented:</p>
<blockquote>
<p>Based on feedback, if you clean install Pro, Enterprise, or Education editions of Windows, the Cortana voice-over will be disabled by default. Screen reader users can still choose to start Narrator at any time by pressing WIN + Ctrl + Enter.</p>
</blockquote>
<p>This means, however, that Cortana will still speak during the initial setup of Windows 10 Home, which should be okay: After all, users of Windows 10 Home will likely only be setting up one or two machines with <a href='//alternativeto.net/software/windows-10/'><img alt='Small Windows 10 icon' class='mini-app-icon' src='//d2.alternativeto.net/dist/icons/windows-10_82838.png?width=36&height=36&mode=crop&upscale=false' />Windows 10</a> at a time, a far cry from the potentially dozens of machines that IT admins may need to set up with Windows 10 Enterprise or Education.</p>
<p>The Windows 10 build with this change, Build 18309, is now available on the Fast ring of the <a href="https://insider.windows.com/en-us/how-to-overview/" rel="nofollow">Windows Insider Program</a>.</p>
