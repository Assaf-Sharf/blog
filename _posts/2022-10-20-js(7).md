# callback function



### 정의

- 함수에 파라미터로 들어가는 함수
- 순차적으로 실행하고 싶을 때 사용



```js
class counter {
  constructor(runevery) {
    this.counter = 0;
    this.callback = runevery;
  }
  increase() {
    this.counter++;
    console.log(this.counter);
    if(this.counter % 5 === 0) {
      this.callback(this.counter);
    }
  }
}
function print() {
  console.log("yo!");
}
const bbcounter = new counter(print);
bbcounter.increase();
bbcounter.increase();
bbcounter.increase();
bbcounter.increase();
bbcounter.increase();

```
