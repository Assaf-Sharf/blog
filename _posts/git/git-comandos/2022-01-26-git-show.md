## Git Show

* Muestra los **metadatos** de ``HEAD`` y las diferencias con el **commit** anterior

```bash
git show
```

* Muestra los **metadatos** del ``último commit`` de la rama ``master`` y las diferencias con el ``commit anterior``

```bash
git show master
```

* Muestra el código del ``<archivo>`` del commit ``HEAD~2``

```bash
git show HEAD~2:<archivo>
```
