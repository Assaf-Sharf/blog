1. 클래스에서 생성자가 정의되어 있지 않은 경우, 컴파일러는 자동으로 생성자 코드를 넣어서 컴파일 한다. Java의 모든 클래스는 자동으로 JDK에서 제공하는 Object 클래스의 서브 클래스가 된다. Object 클래스에는 매개변수가 없는 생성자가 정의되어 있는데 아무 기능도 수행하지 않는 빈 몸체로 구현되어있다. 

```
public class MyClass extends Object {
      public MyClass() {
		super(); //Object의 생성자를 호출 
	}
}
```



컴파일러는 클래스에 생성자가 하나라도 정의되어 있는 경우는 기본 생성자를 만들어주지 않는다. 

```
public class CSuper {
       public CSuper(int a) {
		// TODO Auto-generated constructor stub
    	   System.out.println("super argu");
	}
}
```

슈퍼클래스 CSuper에는 매개변수가 없는 생성자가 없기 때문에 컴파일 오류가 생김.

```
public class Csub extends CSuper {
	public Csub() {
	// TODO Auto-generated constructor stub
		System.out.println("sub non-argu");
}
}
```
