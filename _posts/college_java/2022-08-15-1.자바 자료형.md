## 

1.  정수형 

   java의 정수형에는 byte, short, int, long 이 있으며 각각 1byte, 2byte,4byte,8byte 이다.

    음수는 양수의 2의 보수로 표현된다. (양수의 2진수에서 0을 1로, 1을 0으로 바꾼 1의 보수에서 1을 더하면 2의 보수가 됨)

2.  실수형 

   float (4byte), double(8byte) 

   실수형 리터럴을 직접 사용할 경우 JAVA는 double형 리터럴로 간주함. 

   ```
   float f = 5.0; // 에러가 발생함 
   double d = 5.0;
   ```

3.  문자형 

   char (2byte)가 있음 

   java에서 사용되는 문자 코드로는 1byte 길이의 기존 ASCII가 아니라 2byte길이인 UNICODE를 사용한다. 기존 ASCII로 표현되는 문자 코드는 그대로 하고 길이만 2byte 확장하여 사용되고 비영어권에서 사용되는 문자는 UNICODE코드를 사용하며 예를 들어서 '한'이라는 한글문자는 UNICODE에서 새로운 코드값 54620을 가진다. 

4.  논리형 

   자바에서는 오직 boolean형만이 참 또는 거짓을 표현할 수 있다. 

5. 참조형

   기본형을 제외한 모든 자료형을 참조형이라 한다. 

   ```
   int anArray[]; //배열형 변수
   String szString;// 문자열형의 변수
   ```

6. 자료형의 형변환

   - 묵시적 형변환

   같은 자료형에서, 형변환을 하더라도 자료의 손실이 발생하지 않는 경우

   ```
   float f = 5;
   double db = f; //double형이 float보다 표현범위가 넓기 때문에 오류가 발생하지 않음
   ```

   - 명시적 형변환

     강제로 형변환시키는 경우, 자료 손실이 발생할 수 있음

     ```
     int a = 322550;
     double db2 = a;
     	
     System.out.print(db2);
     ```
