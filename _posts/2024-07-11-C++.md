# C++ 기반의 데이터 입출력 문제
윤성우 저자님의 `열혈C++프로그래밍` 문제 및 해설 정리

1. 사용자로부터 총 5개의 정수를 입력받아서, 그 합을 출력하는 프로그램 작성
```C++
#include<iostream>

int main(void)
{
	int cnt = 1;
	int result = 0;
	int temp;
	for (int i = cnt; i <= 5; i++) {
		std::cout << cnt << "번째 정수 입력: ";
		std::cin >> temp;
		result += temp;
	}
	std::cout << "합계: " << result;


	return 0;
}
```

2. 프로그램 사용자로부터 이름과 전화번호를 문자열의 형태로 입력받아서, 입력받은 데이터를 그대로 출력하는 프로그램
```C++
#include<iostream>

int main(void)
{
	char name[100];
	char phone[30];

	std::cin >> name;
	std::cin >> phone;

	std::cout << name<<std::endl;
	std::cout << phone;


	return 0;
}
```
