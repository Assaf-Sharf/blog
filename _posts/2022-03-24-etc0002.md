<center><h6>html 작성 시 자주 사용되는 타임리프 문법</h6></center>

<div align="center">

```yaml
타임리프 선언 : <html xmlns:th="http://www.thymeleaf.org">

- ${ 변수명 } - 변수 표현식으로 모델에 포함된 값이나, 타임리프 변수로 선언한 값을 조회할 수 있다.

- #{ 변수명 } - 외부 자원 변수 표현식으로 th:text="#{member.register}" 으로 쓰여진다.

```
