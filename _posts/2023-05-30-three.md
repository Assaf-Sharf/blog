# HTML의 기본 구조와 속성

html 버전을 의미하는 태그
```
<!DOCTYPE html>
```
html 문서의 시작을 의미하는 태그는 lang이라는 문서 언어의 정보
```
<html lang= "ko" ></html>
```
태그의 속성이라는 건 태그를 의미나 기능적으로 보조하는 역활을 말합니다.
```
<태그명 속성 = "속성값"></태그명>
```
속성값은 외우지 않아도 공부하다보면 자연스럽게 알게됩니다.
```
<head>
</head>
<body>
</body>
```
head태그는 html 문서의 정보를 작성하는 영역이고, 
```
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
```
body 태그는 웹브라우저에 표시하는 영역입니다.
