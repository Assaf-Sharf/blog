## Numpy 불러오기


```python
import numpy
```

### 1차원 array


```python
array1 = numpy.array([2, 3, 5, 7, 11, 13, 17])
```


```python
type(array1)
```




    numpy.ndarray




```python
print(array1)
```

    [ 2  3  5  7 11 13 17]



```python
array1.shape
```




    (7,)




```python
array1.size # 7개의 요소가 있음
```




    7



### 2차원 array


```python
array2 = numpy.array([[1, 2, 3, 4],[5, 6, 7, 8]])
```


```python
type(array2)
```




    numpy.ndarray




```python
print(array2)
```

    [[1 2 3 4]
     [5 6 7 8]]



```python
array2.shape # array의 세로, 가로길이 즉 행 2개, 열 4개
```




    (2, 4)




```python
array2.size # 2 x 4 = 8개의 요소가 있음
```




    8
