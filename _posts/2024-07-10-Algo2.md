title: "재귀함수-팰린드롬"
---

팰린드롬(palindrome)은 앞으로 읽는 것과 뒤로 읽는 것이 같은 문자열입니다. 
문자열 `my_str`을 받아서 팰린드롬인지를 판별하는 함수 `is_palindrome()`을 작성해보도록 하겠습니다.  

1. 팰린드롬인지 판별
- 팰린드롬인지 판별하기 위해서는 맨 처음 글자와 맨 마지막 글자가 동일한지 판단해야합니다.
  그 다음에는 처음과 마지막 문자가 제거 된 문자열에서도 마찬가지로 맨 처음 글자와 마지막 글자가 동일하다면
  팰린드롬이라고 할 수 있습니다.

2. 베이스 케이스 작성
- 베이스 케이스의 경우 문자열이 한 문자만 갖고 있거나, 빈 문자열이면 팰린드롬이라고 할 수 있습니다. 그래서
- 베이스 케이스는 `len(my_str) <= 1` 이고 `len(my_str) > 1`는 재귀 케이스가 됩니다.

3. 재귀 케이스 작성
- 맨 처음 글자와 맨 마지막 글자가 다르다면 팰린드롬이 아니기 때문에 `False`값을 리턴해줍니다.
- 맨 처음 글자와 마지막 글자가 같다면 팰린드롬이기 때문에 재귀 함수를 호출합니다.


4. 모범 답안

```Python
def is_palindrome(my_str):
    # base case
    if len(my_str) <= 1:
            return True
    
    # 재귀 케이스
    if my_str[0] == my_str[-1]: 
        return is_palindrome(my_str[1:-1])
    else:
        return False


# 테스트 코드
print(is_palindrome('기러기'))
print(is_palindrome('토마토'))
print(is_palindrome('바나나'))
print(is_palindrome('racecar'))
print(is_palindrome('radar'))
print(is_palindrome('stars'))
print(is_palindrome('123321'))

```
