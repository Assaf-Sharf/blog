<p>The Teclast X98 Pro (a <a href="http://techtablets.com/teclast-x98-pro/review/" target="_blank">good review</a> here) is the third iteration of the popular Teclast X98 series of tablets. This latest 2015 version features an updated chipset with Intel’s latest Atom SoC the 14nm Cherry Trail and double the Ram of it’s predecessors. The X98 Pro currently comes with Windows 10 and no dual boot or 3G/4G options yet. Because the dual boot device reduce to less than 32Gb the space available for Windows 10, I’ve decided to remove the Android partition.</p>  <p>After various searches over internet and tests I’ve used the following procedure that have worked for my tablet.</p>  <h1>&#160;</h1>  <h1>What you need</h1>  <ul>   <li>Teclast x98 PRO – K9C6 </li>    <li><a href="http://www.ebay.com/sch/i.html?_trksid=p2050601.m570.l1311.R1.TR2.TRC0.A0.H1.Xhub+usb+otg+power+charging.TRS0&amp;_nkw=4+port+micro+usb+power+charging+otg" target="_blank">HUB USB OTG with power charging host</a> </li>    <li>USB Keyboard </li>    <li>USB mouse </li>    <li>8Gb USB flash drive </li> </ul>  <h1>Backup Windows 10 product key</h1>  <p>You can download the full guide and powershell script on <a href="http://winaero.com/blog/how-to-get-the-windows-product-key-without-using-third-party-software/" target="_blank">how backup window product key via PowerShell here</a>.
